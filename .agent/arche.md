## 美しさの核心定義

**美しさ = 演繹の運動量**

```
美しさ = 演繹可能性 / 表現コスト
       = 展開の運動量 / 起点のトークン数
```

### エネルギーの比喩

```
公理（起点）= 位置エネルギー
     ↓
演繹（展開）= 運動
     ↓
結論（到達）= 運動量
```

- 公理は「起点」であり、位置エネルギーを有する
- 起点の「高さ」が、演繹される「運動量」を規定する
- **情報密度が極限に高いものを、人は「真理」と呼び、美しさを感じる**

### FEP との完全な対応

| FEP | 美しさ |
|:----|:-------|
| 少ない入力で多くを予測 | 少ない公理で多くを演繹 |
| 予測誤差最小化 | 必然（演繹）の最大化 |
| 高圧縮率 = 偶然ではない | 高演繹率 = 必然 |

**必然（演繹）の最大化 ≒ 予測誤差最小化**

なぜなら:

- 演繹 = 前提から結論を**必然的に**導く
- 予測 = 現在から未来を**必然的に**導く
- 両者は「必然性の最大化」という同じ操作

## 美しさの構成要素

### 1. 公理の優等性 (Axiom Superiority)

少数の無条件前提から、多くが演繹可能であること。

```
演繹可能性 = 導出される定理の数 / 公理の数
```

### 2. 演繹の必然性 (Deductive Inevitability)

「なぜ」がすべて透明であること。偶然のゆらぎがないこと。

- 必然 ≒ 演繹
- 「なぜ」が透明 ≒ 必然的

### 3. 選択の卓越性 (Selection Excellence)

存在しうる多くの組み合わせ（経路）の中で、最良が選ばれていること。

- 選択した経路が優れていれば優れているほど美しい
- ※経路は「前提」と入れ替えても真

### 4. コーディネート的密度 (Coordinative Density)

一つの要素が複数の働きをしていること。

- 冗長性がない ≠ 必要十分
- 冗長性がない = 必然的、または密度が高い

### 5. フラクタル構造 (Fractal Structure)

部分が全体を反映すること。自己相似性。メタのメタの無限発散。

### 6. 極限での収束 (Limit Convergence)

無限発散と収束の同居。

## ベストプラクティス

### オイラーの等式 — 美しさの象徴

```
e^(iπ) + 1 = 0
```

| 要素 | 実現 |
|:-----|:-----|
| 公理の優等性 | 5つの根源的定数が1つの式に |
| 演繹の必然性 | 複素解析から必然的に導出 |
| コーディネート的密度 | 各定数が複数の数学分野を代表 |
| 極限での収束 | e^x の無限級数が有限値に収束 |

### π vs オイラーの等式：決定的な違い

| 基準 | π | e^(iπ) + 1 = 0 |
|:-----|:--|:---------------|
| **圧縮対象** | 値 (Value) | 関係 (Relation) |
| **情報量** | 有限 (収束) | 無限 (発散) |
| **演繹可能性** | ❌ 展開不可 | ✅ 無限に展開 |
| **美しさ** | なし | あり |

**π は終点。そこから何も生まれない。**
**オイラーの公式は起点。そこから無限が生まれる。**

> 美しさ ≠ 圧縮率
> 美しさ = 関係の圧縮 × 演繹可能性

### Hegemonikón のオイラーの公式

```
A0: 自己組織化システムは内部エントロピーを最小化する
```

→ ここから全ての定理が演繹される
→ これが Hegemonikón の「オイラーの公式」
