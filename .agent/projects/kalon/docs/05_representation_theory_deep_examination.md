## 2. 表現論の要件

### 群の表現とは

群 G のベクトル空間 V 上の表現:

```
ρ: G → GL(V)

ρ(g₁ · g₂) = ρ(g₁) · ρ(g₂)  (準同型)
ρ(e) = I                      (単位元 → 恒等行列)
```

### 既約表現とは

表現 (V, ρ) が既約 ⟺ V の ρ-不変な部分空間が {0} と V のみ。
= 「これ以上分解できない最小の表現」。

### ρ₃: 定理 → 派生 (∞)

```
各定理に +, -, *, >> 等の演算子を適用 → 派生 WF
/noe → /noe+, /noe-, /noe++, ...
```

**派生は「定理の作用による変換」:**

- - = 精度パラメータの上昇 (自然変換 η: Id ⟹ T)
- 各派生は、元の定理に自然変換を適用した結果

**既約表現 = 原子的派生?**

派生の「分解不能性」:

- `/noe+` = `/noe` に `+` を 1 回適用 → 原子的（これ以上分解すると /noe と + に戻る）
- `/noe++` = `/noe` に `+` を 2 回適用 → 分解可能（/noe+ に + を適用）

**判定: ⚠️ 部分的。** 派生が「既約表現」であるためには、表現する群と空間が明確に定義されている必要がある。`+` の繰り返し適用が群構造を持つかは Field 2 で否定済み（Z/4Z ではない）。

ただし `+` を自然変換 (Field 1) として解釈すれば:

- **原子的派生 = 自然変換の 1 回適用** = 関手圏における射
- **複合派生 = 自然変換の合成** = 関手圏における射の合成
- **「既約」= 自然変換として分解不能** → これは意味が通る
