# 二項対立の必然性

> **区別の最小形式は二項対立である。これは存在の必然的構造である。**

---

## 論証

### 1. 区別の最小形式

```
区別 (distinction) = X と not-X
```

- 1つ = 区別なし = 情報なし = 無
- 2つ = 最小の区別 = 最小の情報
- 3つ以上 = 二項区別の入れ子

**結論**: 区別の最小単位は二項対立

---

### 2. 圏論的解釈

```
最小の非自明な圏 = 2対象 + 1射

    0 ──f──> 1
```

- 1対象のみ = 恒等射のみ = 自明
- **関係が存在するためには「相手」が必要**
- 他（外）があるから自（内）が生まれる

---

### 3. 情報理論

```
情報の最小単位 = bit = {0, 1}
情報量 = log₂(選択肢の数)
最小の非自明な情報 = log₂(2) = 1 bit
```

---

### 4. FEP からの選好

```
エントロピー最小化 = 最小の区別で最大の予測
区別が多すぎる = エントロピー増加 = FEP 違反
最適解 = 最小の区別 = 二項
```

---

### 5. n項の還元可能性

```
A, B, C の三項区別
  = (A vs not-A) ∧ (B vs not-B within not-A)
  = 二項対立の入れ子
```

**n項区別は (n-1) 回の二項区別に還元可能**
