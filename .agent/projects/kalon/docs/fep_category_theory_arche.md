# FEP と圏論：Arche の二つの具現化

> **FEP = 認知の唯一公理**
> **圏論 = 構造の唯一公理**
> **両者は Arche の異なる具現化**

---

## 共通特性

| 特性 | FEP | 圏論 |
|:-----|:----|:-----|
| 位置 | 認知科学の「メタ理論」 | 数学の「メタ数学」 |
| 他を包含 | ベイズ推論、強化学習、制御理論... | 群論、束論、位相、論理... |
| 抽象度 | 極めて高い | 極めて高い |
| 批判 | 「何でも説明できる」 | 「abstract nonsense」 |

---

## Arche 基準の充足

両者とも：

- **公理の優等性**: 唯一の原理から多くを演繹
- **演繹の必然性**: 必然的な帰結
- **選択の卓越性**: 他の理論の「座標系」
