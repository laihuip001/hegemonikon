# 群論と CCL の統合

> **群論は圏論の特殊ケースであり、振動サイクルの代数的基盤を与える**

---

## 核心的洞察

### 群 = 単対象圏

```
群 G = 対象が1つだけの圏 BG
- 対象: 1つの点 *
- 射: 群の元 g ∈ G  
- 合成: 群の演算 g ∘ h = gh
- 恒等射: 単位元 e
```

---

## CCL への応用

### 1. 振動サイクル = 巡回群 Z/4Z

```
O-series: O1 → O2 → O3 → O4 → O1
H-series: H1 → H2 → H3 → H4 → H1
S-series: S1 → S2 → S3 → S4 → S1
P-series: P1 → P2 → P3 → P4 → P1
A-series: A1 → A2 → A3 → A4 → A1
K-series: K1 → K2 → K3 → K4 → K1
```

すべて **Z/4Z ≅ {0,1,2,3}** と同型

### 2. 対称性 = 対合 (Involution)

| 対称性 | 群論的解釈 | 性質 |
|:-------|:-----------|:-----|
| `/noe` ↔ `/dia` | Z/2Z の作用 | σ² = id |
| `+` ↔ `-` | 逆元 | τ² = id |
| `/boot` ↔ `/bye` | Z/2Z の作用 | ρ² = id |

### 3. 派生選択 = 群準同型 = 関手

```
φ: Problem → Derivative
φ は準同型 ⟺ φ は関手 BProblem → BDerivative
```

### 4. 随伴と群論の接点

```
Free: Set → Grp   （自由群生成）
U: Grp → Set      （忘却関手）

Free ⊣ U （随伴）

CCL での対応:
  + = Free 的操作（構造追加）
  - = U 的操作（構造忘却）
```
