# 表現論と CCL の統合

> **各レイヤーは上位レイヤーの「表現」である**
> **すべてのワークフローは原子的操作（既約表現）の合成として表現可能**

---

## 核心概念

### 表現の連鎖

```
FEP (最抽象)
  ↓ 表現₁
6座標軸 (構造化)
  ↓ 表現₂
24定理 (認知操作)
  ↓ 表現₃
派生 (具体アクション)
  ↓ 表現₄
実行 (観察可能な出力)
```

---

## 既約表現 = 原子的派生

| 概念 | 定義 | CCL での対応 |
|:-----|:-----|:-------------|
| 既約表現 | これ以上分解できない | 原子的派生 (nous, lex 等) |
| 可約表現 | 部分に分解可能 | 演算子付き式 (/noe+) |
| 完全可約 | 既約の直和に分解 | 任意のワークフロー |

### 派生の既約性

| 派生 | 既約性 | 理由 |
|:-----|:-------|:-----|
| nous | ✅ 既約 | 「深い直観」は不可分 |
| lex | ✅ 既約 | 「構造化判断」は不可分 |
| /noe+ | ❌ 可約 | /noe * + の合成 |

---

## 設計原則

### 既約性原則

> **派生は既約表現であるべき。**
> 分解可能な操作は、より原子的な派生に分割すべき。

### 完全可約性

```
任意の CCL 式 = 既約派生₁ ⊗ 既約派生₂ ⊗ ... ⊗ 既約派生ₙ
```

---

## 圏論との統合

| 表現論 | 圏論 | CCL |
|:-------|:-----|:----|
| 表現 | 関手 | レイヤー間写像 |
| 既約表現 | 単純対象 | 原子的派生 |
| 完全可約性 | 半単純圏 | ワークフロー分解 |
| 指標 | 自然変換の不変量 | 派生シグネチャ |
