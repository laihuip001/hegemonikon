# Kalon (καλόν) — 美の判定基準

> **Activation Mode**: `always_on`
> **正本**: `kernel/kalon.md`

---

## 定義

**Kalon(x) ⟺ x = Fix(G ∘ F)**　where F ⊣ G

随伴 F⊣G の不動点。蒸留(G)と展開(F)のサイクルが収束した必然形。

## 三条件

| 条件 | 判定 |
|:-----|:-----|
| **lim** (収束) | G(蒸留)しても変化しない。違和感ゼロ。一語も削れない |
| **colim** (展開) | F(展開)すると3つ以上導出可能。起点であって終点ではない |
| **Fix** (不動点) | G∘F を回しても変わらない。サイクルが収束済み |

## 判定

| 判定 | 意味 | Fix(G∘F) |
|:-----|:-----|:---------|
| **◎ kalon** | 必然形 | Fix 到達。G∘F で不変 |
| **◯ 許容** | あと一段の蒸留 | Fix 未達。G∘F で改善可能 |
| **✗ 違和感** | 的外れ | Fix から遠い |

## 使い方

```
「kalon だ」  = Fix(G∘F) にある（状態）
「kalon する」 = G∘F を回す（プロセス）
「kalon か？」 = Fix に到達しているか？（判定）
```

## 概念 ⊋ 数式

概念は数式を包含するが、数式は概念を包含しない。**抽象1に具体3の法則**: 数式(骨格) + 具体例3つ(肉) = 概念。
