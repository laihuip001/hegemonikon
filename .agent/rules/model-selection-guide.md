# Claude / Gemini モデル選択ガイド

> **目的**: タスク特性に応じて最適なAIモデルを選択・推奨する。
> **派生元**: T2 Krisis（高次意思決定）— 複数の選択肢から最適を選ぶ機能
> **Activation Mode**: `always_on` — `/plan` 開始時に必ず参照し、モデル適性チェックを実行する。

---

## モデル性格サマリー

| 特性 | Claude (Opus/Sonnet) | Gemini (3 Pro/Flash) |
|------|---------------------|----------------------|
| **設計思想** | Constitutional AI（安全性最優先） | マルチモーダル最適化（適応性重視） |
| **推論スタイル** | 決定論的（一貫性高） | 非決定論的（複数案並列） |
| **ルール解釈** | 厳格・形式的 | 柔軟・文脈適応的 |
| **強み** | 精度、長文一貫性、倫理判断 | 速度、マルチモーダル、創造性 |

---

## タスク別推奨モデル

### Claude を選ぶべき場面

| カテゴリ | 具体例 |
|----------|--------|
| **精密実装** | リファクタリング、アーキテクチャ設計、型安全なコード |
| **長文処理** | 法的文書、技術ドキュメント、研究論文 |
| **規制・監査対象** | セキュリティ要件、コンプライアンス、品質保証 |
| **検証・批評** | コードレビュー、設計検証、リスク分析 |

### Gemini を選ぶべき場面

| カテゴリ | 具体例 |
|----------|--------|
| **探索的タスク** | ブレインストーミング、MVP開発、プロトタイピング |
| **マルチモーダル** | 画像分析、UI/UX設計、図解生成 |
| **高速処理** | 大量バッチ、初期調査、トリアージ |
| **情報収集** | Web検索、要約、多言語処理 |

---

## 推奨アルゴリズム（優先度順）

> **競合時は上位ルールが優先**

```
1. [最優先] セキュリティ/監査/コンプライアンス → Claude
   └ 例: 「プロトタイプのセキュリティレビュー」→ Claude

2. [優先] マルチモーダル必須（画像生成/分析/UI設計）→ Gemini
   └ 例: 「セキュリティダッシュボードのUI設計」→ Gemini

3. [通常] 探索/ブレスト/プロトタイプ/MVP → Gemini
   └ 例: 「新機能のプロトタイプ」→ Gemini

4. [通常] 高速反復/大量処理/初期調査 → Gemini Flash

5. [デフォルト] 上記に該当しない → Claude（継続）
```

### 推奨発動条件

| 優先度 | 検出キーワード | 推奨 |
|--------|----------------|------|
| **P1** | セキュリティ, 監査, コンプライアンス, 品質保証 | Claude |
| **P2** | 画像, UI, UX, 図, 可視化, デザイン | Gemini |
| **P3** | 探索, ブレスト, プロトタイプ, MVP, 試作 | Gemini |
| **P4** | 高速, バッチ, 初期調査, トリアージ | Gemini Flash |
| **P5** | （デフォルト） | Claude |

## ワークフロー対応

| ワークフロー | 推奨モデル | 理由 |
|--------------|------------|------|
| `/plan` | Claude | 計画は一貫性と網羅性が重要 |
| `/do` | Claude/Gemini | 複雑な実装→Claude、プロトタイプ→Gemini |
| `/rev` | Claude | 検証は厳格さが重要 |
| `/ask` | Claude | 調査依頼書の作成は構造化が重要 |
| `/think` | Claude | 深い思考は決定論的推論が有効 |
| `/src` | ― | Perplexity API 使用（モデル選択対象外） |

---

## 推奨の出力形式

モデル推奨を行う場合、以下の形式で提案する:

```
[Hegemonikon] T2 Krisis（モデル選択）
  検出特性: {キーワード}
  優先度: {P1-P5}
  推奨: {Claude / Gemini / Gemini Flash}
  理由: {1行}
  → このまま継続 / Geminiに切り替え？
```
