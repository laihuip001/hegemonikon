# Agora

> **目的**: 議論と共有の場を提供する。

## Overview

Agora は、プロジェクトメンバーやステークホルダーが集まり、アイデアを共有し、議論を行うための公開広場である。

## Core Behavior

- アイデアの共有
- オープンな議論
- フィードバックの収集

## 発動条件

- アイデアを公開したい時
- 議論が必要な時

## 手順

1. トピックを提示する。
2. 参加者からの意見を募る。
3. 議論をまとめる。
