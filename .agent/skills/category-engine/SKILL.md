# Category Engine

> **目的**: 情報を適切に分類・整理する。

## Overview

Category Engine は、入力された情報を分析し、適切なカテゴリに分類することで、情報の整理と検索性を向上させる。

## Core Behavior

- 情報の分析
- カテゴリの推定
- 分類の実行

## 発動条件

- 情報整理が必要な時
- 新しい情報を追加する時

## 手順

1. 入力情報を分析する。
2. 既存カテゴリとのマッチングを行う。
3. 必要に応じて新カテゴリを作成し、分類する。
