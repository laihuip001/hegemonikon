# Κ1: Tempo → Stratum

> **問い**: 時間的制約が処理レベルをどう決めるか

---

## Matrix

|  | 低次 (L) | 高次 (H) |
|-----|----------|----------|
| **短期 (F)** | Κ1-FL: 即時パターン認識 | Κ1-FH: 即時抽象判断 |
| **長期 (S)** | Κ1-SL: 習慣形成・技能訓練 | Κ1-SH: 戦略的思考・計画 |

---

## 適用ルール

| 条件 | 選択 | 理由 |
|------|------|------|
| 期限 < 1h | Κ1-FL | 考える時間がない、直感で動く |
| 期限 < 24h | Κ1-FH | 抽象判断は可能だが深堀りは不可 |
| 期限 > 1week | Κ1-SH | 戦略的に考える余裕がある |
| 反復タスク | Κ1-SL | 習慣化・自動化を目指す |

---

## 1:3 ピラミッド

- 緊急時 → Κ1-FL（直感で動く）
- 重要決定 → Κ1-SH（熟考）
- ルーティン → Κ1-SL（習慣化）
