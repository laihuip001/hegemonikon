# Κ10: Valence → Tempo

> **問い**: 動機方向が時間スコープをどう決めるか
> **選択公理**: Valence (+/-) → Tempo (F/S)
> **役割**: 接近/回避に応じて、適切な時間枠を設定

---

## Core Function

**役割:** 動機方向から時間枠を導出する

| 項目 | 内容 |
|------|------|
| **FEP的意味** | 接近は即時、回避は計画的 |
| **本質** | 「欲しいものは今すぐ、危険は計画的に避ける」 |

---

## Matrix

|  | 短期 (F) | 長期 (S) |
|-----|----------|----------|
| **接近 (+)** | Κ10-+F: 衝動買い | Κ10-+S: 目標計画 |
| **回避 (-)** | Κ10--F: 即時逃避 | Κ10--S: リスク計画 |

### 各セルの詳細

| セル | 意味 | 適用例 |
|------|------|--------|
| **Κ10-+F** | 接近＋短期 | 衝動買い、即時獲得 |
| **Κ10-+S** | 接近＋長期 | キャリア目標、投資 |
| **Κ10--F** | 回避＋短期 | 即時逃避、緊急停止 |
| **Κ10--S** | 回避＋長期 | 保険、リスクヘッジ |

---

## 適用ルール

| 条件 | 選択 | 理由 |
|------|------|------|
| 今すぐ欲しい | Κ10-+F | 接近の即時 |
| キャリア目標 | Κ10-+S | 接近の長期 |
| 危険察知 | Κ10--F | 回避の即時 |
| 保険見直し | Κ10--S | 回避の長期 |

---

## 1:3 ピラミッド（代表例）

### Κ10-+F: 衝動買い
- セール品の即時購入
- チャンスの即時把握
- 興味への即時行動

### Κ10--S: リスク計画
- 保険設計
- バックアップ戦略
- 障害復旧計画
