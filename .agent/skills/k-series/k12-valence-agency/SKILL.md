# Κ12: Valence → Agency

> **問い**: 動機方向が主体選択をどう決めるか
> **選択公理**: Valence (+/-) → Agency (S/E)
> **役割**: 接近/回避に応じて、自己か環境かを決定

---

## Core Function

**役割:** 動機方向から「誰を/何を変えるか」を導出する

| 項目 | 内容 |
|------|------|
| **FEP的意味** | 接近/回避の対象が自己か環境か |
| **本質** | 「何に向かうか、何から離れるか」 |

---

## Matrix

|  | 自己 (S) | 環境 (E) |
|-----|----------|----------|
| **接近 (+)** | Κ12-+S: 学習意欲 | Κ12-+E: 関係構築 |
| **回避 (-)** | Κ12--S: 自己否定 | Κ12--E: 有害関係排除 |

### 各セルの詳細

| セル | 意味 | 適用例 |
|------|------|--------|
| **Κ12-+S** | 接近＋自己 | 新スキル習得、自己投資 |
| **Κ12-+E** | 接近＋環境 | 人脈拡大、良い環境づくり |
| **Κ12--S** | 回避＋自己 | 悪習を断つ、弱点克服 |
| **Κ12--E** | 回避＋環境 | 有害関係から離れる |

---

## 適用ルール

| 条件 | 選択 | 理由 |
|------|------|------|
| 学習意欲 | Κ12-+S | 自己への接近 |
| ネットワーク | Κ12-+E | 環境への接近 |
| 悪習断ち | Κ12--S | 自己からの回避 |
| 有害関係 | Κ12--E | 環境からの回避 |

---

## 1:3 ピラミッド（代表例）

### Κ12-+S: 学習意欲
- 新技術の学習
- 資格取得
- 健康増進

### Κ12--E: 有害関係排除
- 有害な依存関係の削除
- 毒性のある環境からの離脱
- レガシーからの脱却
