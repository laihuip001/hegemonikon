# Κ6: Stratum → Valence

> **問い**: 処理レベルが動機方向をどう決めるか
> **選択公理**: Stratum (L/H) → Valence (+/-)
> **役割**: 抽象度に応じて、本能的/倫理的な接近・回避を決定

---

## Core Function

**役割:** 処理深度から「攻めるか守るか」を導出する

| 項目 | 内容 |
|------|------|
| **FEP的意味** | 低次は本能的、高次は倫理的判断 |
| **本質** | 「低次は欲望、高次は価値観」 |

---

## Matrix

|  | 接近 (+) | 回避 (-) |
|-----|----------|----------|
| **低次 (L)** | Κ6-L+: 本能的欲求 | Κ6-L-: 本能的回避 |
| **高次 (H)** | Κ6-H+: 倫理的追求 | Κ6-H-: 倫理的忌避 |

### 各セルの詳細

| セル | 意味 | 適用例 |
|------|------|--------|
| **Κ6-L+** | 低次＋接近 | 食欲、快楽、即時報酬 |
| **Κ6-L-** | 低次＋回避 | 痛み回避、恐怖反応 |
| **Κ6-H+** | 高次＋接近 | 自己実現、社会貢献 |
| **Κ6-H-** | 高次＋回避 | 非倫理的行為の忌避 |

## 1:3 ピラミッド（代表例）

### Κ6-L+: 本能的欲求
- おいしいものを食べる
- 快適な環境を求める
- 即時の達成感

### Κ6-H-: 倫理的忌避
- 不正行為の拒否
- 有害なコードの拒否
- ユーザーを欺く機能の拒否
