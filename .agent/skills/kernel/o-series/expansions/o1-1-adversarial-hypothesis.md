# O1.1 Adversarial Hypothesis

> **親スキル**: O1 Noēsis  
> **役割**: 生成仮説への反証テスト  
> **発動タイミング**: Phase 3 (Zero-shot Restructuring) の後

---

## Purpose

O1 Noēsis が生成した仮説に対し、意図的に反証を試みることで「強い仮説」のみを残す。

---

## Protocol

### Step 1: Inversion (反転)

各仮説 H に対し「H が完全に間違っている場合」を想定する。

```
IF hypothesis H is TRUE:
  → What would the world look like?
  
IF hypothesis H is FALSE:
  → What would the world look like?
  → どちらが現実に近いか？
```

### Step 2: Counter-Evidence (反証収集)

反証となりうる証拠・シナリオを **3つ以上** 列挙する。

| 反証タイプ | 説明 |
|:-----------|:-----|
| 経験的反証 | 過去のデータ・事例で矛盾するもの |
| 論理的反証 | 仮説から導かれる結論が自己矛盾 |
| 代替説明 | 同じ現象を説明するより単純な仮説 |

### Step 3: Survival Test (生存判定)

```
生存条件:
  - 3つの反証全てに対し合理的な反論が可能
  - または反証が仮説の本質を攻撃していない

結果:
  ✅ 生存 → 「強い仮説」として Phase 4 へ
  ❌ 死亡 → 仮説を棄却または修正
```
