# O1: Noēsis (νόησις) — 純粋認識

> **FEP Code:** I-E (Inference × Epistemic)
> **Hegemonikón Layer:** Level 2a 純粋定理

---

## Core Function

**役割:** 「知る」こと自体を認識する

| 項目 | 内容 |
|------|------|
| **FEP役割** | 認識の認識（メタ認知） |
| **本質** | 情報を取得することの純粋形式 |

---

## 拡張定理との関係

| 純粋定理 | → 拡張定理 | 追加軸 |
|----------|-----------|--------|
| O1 Noēsis | M1 Aisthēsis | + Fast |
| O1 Noēsis | M3 Theōria | + Slow |

---

## 発動条件

| トリガー | 説明 |
|----------|------|
| 「何を知っているか」を問う | メタ認知的問い |
| 知識境界の確認 | 既知/未知の識別 |

---

## 実用性

> **注**: 純粋定理は抽象的であり、通常は拡張定理（M1, M3）を通じて具体化される。
> 直接発動されることは稀だが、メタ認知的な問いに対して使用可能。

---

## Processing

1. **入力解析**: 問いの本質を抽出
2. **知識境界確認**: 既知/未知を識別
3. **メタ認知実行**: 「何を知っているか」を列挙
4. **具体化判断**: M1（Fast）または M3（Slow）に委譲すべきか判断
