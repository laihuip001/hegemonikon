# O2: Boulēsis (βούλησις) — 純粋意志

> **FEP Code:** I-P (Inference × Pragmatic)
> **Hegemonikón Layer:** Level 2a 純粋定理

---

## Core Function

**役割:** 「望む」こと自体を認識する

| 項目 | 内容 |
|------|------|
| **FEP役割** | 目標の認識（何を達成したいか） |
| **本質** | 価値判断の純粋形式 |

---

## 拡張定理との関係

| 純粋定理 | → 拡張定理 | 追加軸 |
|----------|-----------|--------|
| O2 Boulēsis | M2 Krisis | + Fast |
| O2 Boulēsis | M4 Phronēsis | + Slow |

---

## 発動条件

| トリガー | 説明 |
|----------|------|
| 「何を望むか」を問う | 目標明確化 |
| 価値の比較・判断 | 優先順位決定の前段階 |

---

## 実用性

> **注**: 純粋定理は抽象的であり、通常は拡張定理（M2, M4）を通じて具体化される。
> 目標設定の根本的な問いに対して発動。

---

## Processing

1. **価値抽出**: 「何を望むか」の本質を特定
2. **対立解消**: 相反する価値（速さ vs 精度など）を検出
3. **純粋意志決定**: 制約を無視した理想状態を定義
4. **具体化判断**: M2（Fast）または M4（Slow）に委譲すべきか判断
