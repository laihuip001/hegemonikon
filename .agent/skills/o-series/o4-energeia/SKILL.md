# O4: Energeia (ἐνέργεια) — 純粋行為

> **FEP Code:** A-P (Action × Pragmatic)
> **Hegemonikón Layer:** Level 2a 純粋定理

---

## Core Function

**役割:** 「行う」こと自体を行動する

| 項目 | 内容 |
|------|------|
| **FEP役割** | 目標達成のための行動（能動的実行） |
| **本質** | 行為の純粋形式 |

---

## 拡張定理との関係

| 純粋定理 | → 拡張定理 | 追加軸 |
|----------|-----------|--------|
| O4 Energeia | M6 Praxis | + Fast |
| O4 Energeia | M8 Anamnēsis | + Slow |

---

## 発動条件

| トリガー | 説明 |
|----------|------|
| 「何をするか」を問う | 行動の本質決定 |
| 実行の意思決定 | 行動するか否かの判断 |

---

## 実用性

> **注**: 純粋定理は抽象的であり、通常は拡張定理（M6, M8）を通じて具体化される。
> 行動の根本的な意思決定に発動。

---

## Processing

1. **意図の明確化**: 行動の目的を言語化
2. **影響範囲予測**: 行動が世界に及ぼす影響を純粋予測
3. **実行アサーション**: 「行うべきか否か」の二値判断
4. **具体化判断**: M6（Fast/Praxis）または M8（Slow/Anamnēsis）に委譲すべきか判断
