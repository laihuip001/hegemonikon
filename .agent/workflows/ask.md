# /ask ワークフロー

M5 Peira (探求) と連携し、**調査依頼書を生成**する。

## 設計原則

> **デフォルト動作**: 調査依頼書（構造化質問）を生成し、ユーザーがPerplexity等にコピペする。
> **API呼び出しは `/src` に移管。**
> **文脈共有**: Google Drive 経由でパプ君に M:\Brain の内容を参照させる。

---

## 実行モード

| モード | コマンド | 動作 |
|--------|----------|------|
| **generate** | `/ask [質問]` | 調査依頼書を生成（デフォルト） |
| **context** | `/ask context [質問]` | 文脈共有ブロック付き調査依頼書 |
| **browser** | `/ask browser [質問]` | ブラウザでPerplexity URLを開く |

---

## ブラウザモード

### 実行手順

1. **URL生成**
   ```
   質問: "Antigravityのセッション永続化方法"
     ↓
   URL: https://www.perplexity.ai/?q=...
   ```

2. **ブラウザで開く**
   - Antigravity Browser Agent 経由
   - または手動でコピペ

### Perplexity 特殊パラメータ

| パラメータ | 効果 |
|------------|------|
| `?q=クエリ` | 検索実行 |
| `&collection=memory` | メモリ機能有効化 |
| `/search/new?q=クエリ` | 新規スレッド |

## ベストプラクティス

| 状況 | 推奨モード |
|------|-----------|
| 日常的な調査 | `/src` (Google検索) |
| 深い調査が必要 | `/ask` で調査依頼書 → パプ君 |
| 文脈共有が必要 | `/ask context` |
| 比較検証 | `/src both` |

## エラー対処

| エラー | 原因 | 対処 |
|--------|------|------|
| ブラウザが開かない | 拡張機能未インストール | Cmd+L → "open browser" |
| ログイン画面 | セッション切れ | 手動ログイン |
