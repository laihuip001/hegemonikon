# /bou: 意志明確化ワークフロー (Boulēsis)

> **Hegemonikón Module**: O2 Boulēsis（意志・欲求）
> **目的**: 「何を望むか」を問い、目的を明確化する
> **発動条件**: プロジェクト開始時、方向性に迷った時、作業が一段落した時

---

## 本質

> βούλησις (Boulēsis) = 熟慮された欲求、理性に基づいた意志

単なる衝動ではなく、「本当に何を望むか」を問う。

---

## 発動条件

| トリガー | 説明 |
|----------|------|
| `/bou` | 意志明確化を開始 |
| `/bou [目標]` | 特定の目標について意志を問う |
| 作業の一段落 | 「次は何をしたいか」 |
| 方向性の迷い | 「そもそも何を望んでいるか」 |

## Process

「なぜそれを達成したいか」（5 Whys）

「本当に望んでいるのはどれか」

「最も重要なものは何か」
