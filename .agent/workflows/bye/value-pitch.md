# Value Pitch — Step 3.6π

> **圏論**: R^π: Ses → Sig — セッションから意義 (Significance) を抽出する関手。
> R (Handoff: 事実の圧縮) とは独立。同じ圏 Ses から異なる圏への射影。
>
> **本質**: 「で、なんなの？」を構造的に答える。
> 技術的事実 (聖) を行為可能性 (俗) に降ろす変換。

---

## Benefit Angle — 行為可能性の8次元

> **導出**: HGK 7公理 + X-series から演繹。
> 各角度は「Creator の affordance がどの次元で拡大したか」を宣言する。

| # | 公理 | 角度 | 問い | 感覚 |
|:--|:-----|:-----|:-----|:-----|
| 1 | FEP | **わかる** | なぜ因果が見えるようになったか | 霧が晴れた |
| 2 | Flow | **できる** | なぜ不可能が可能になったか | 壁を越えた |
| 3 | Value | **深い** | なぜ構造を貫く原理か | 骨まで見えた |
| 4 | Scale | **軽い** | なぜ貴方の負担を減らすか | 重荷が下りた |
| 5 | Function | **育つ** | なぜ未来の貴方を助け続けるか | 種を蒔いた |
| 6 | Valence | **守る** | なぜ貴方を○○から守るか | 盾ができた |
| 7 | Precision | **確か** | なぜ根拠を持って語れるか | 地に足がつく |
| + | X-series | **響く** | なぜ掛け合わさって効くか | 偶然じゃない |

### 合成角度

複数の基底角度が同時に高まると**創発する**性質:

| 合成角度 | 基底 | いつ使うか |
|:---------|:-----|:-----------|
| **美しい** | わかる × 軽い | 構造が明晰で無駄がない。FEP精度↑ × 複雑性↓ |
| **人間的** | 確か × 深い | 根拠が思想に接地し共感を呼ぶ |

> 合成角度をタイトルに使ってよいが、本文では基底角度に分解して語る。

## 角度の選び方

```
成果を見て自問:
  「Creator はこの成果のおかげで、何が《できるように/楽に/安全に》なったか？」

→ 因果が見えた、説明できた      → わかる (FEP)
→ 不可能が可能になった          → できる (Flow)
→ 原理・構造を掴んだ            → 深い  (Value)
→ 摩擦が消えた、負荷が減った    → 軽い  (Scale)
→ 未来の自分が助かる仕組み      → 育つ  (Function)
→ 脅威・劣化から守られた        → 守る  (Valence)
→ 根拠を持って語れる            → 確か  (Precision)
→ 成果同士が掛け合わさった      → 響く  (X-series)

→ 複合で「美しい」「人間的」と感じたら:
  → 基底角度を2つ特定してからタイトルに使う
```

## セルフチェック（記入後に自問）

1. **Creator が明日 /boot したとき、この成果のおかげで何が違うか？**
   → 答えが「特に変わらない」なら売り込みが弱い
2. **もしこの作業をしていなかったら、3ヶ月後に何が起きていたか？**
   → 答えが曖昧なら「もしやらなかったら」が弱い
3. **Hegemonikón を知らない人に30秒で説明するなら？**
   → 説明できないなら俗語への降下が足りない
