# /ccl-dig: 深掘りマクロ

> **CCL**: `@dig = /s+~(/p*/a)_/dia*/o+`
> **用途**: 多面的に深く考えたいとき

## 展開

1. `/s+` — 戦略を詳細化
2. `~(/p*/a)` — 環境×判断と振動
3. `_/dia*/o+` — 判定×認識を融合

## 使用例

```ccl
@dig              # 標準的深掘り
@dig+             # さらに深く（全WFを+で展開）
@dig _ /ene       # 深掘り後、実行へ
```
