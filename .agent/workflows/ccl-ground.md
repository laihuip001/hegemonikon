# /ccl-ground: 具体化マクロ

> **CCL**: `@ground = @scoped{/tak-}*@partial{/bou+, mode:6w3h}~/p-_/ene-`
> **用途**: 抽象概念を具体的タスクに落とす

## 展開

1. `@scoped{/tak-}` — 軽量タスク整理のスコープを限定
2. `*@partial{/bou+, mode:6w3h}` — 意志に 6W3H モードを部分適用して融合
3. `~/p-` — 環境条件と振動
4. `_/ene-` — 軽量実行へ接続

## 使用例

```ccl
@ground                    # 標準具体化
@ground+                   # 全WFを+で展開
@ground _ @build           # 具体化後に構築
```

## CPL 構文

`@scoped`, `*`, `@partial`, `~`, `_`
