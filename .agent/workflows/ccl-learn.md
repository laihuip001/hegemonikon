# /ccl-learn: 学習永続化マクロ

> **CCL**: `@learn = /pro_/dox+_F:[×2]{/u+~(/noe*/dia)}_~(/h*/k)_/pis_/bye+`
> **用途**: 今の学びを次のセッションに残したいとき
> **認知骨格**: Prior → Likelihood → Posterior

## 展開

| 相 | ステップ | 意味 |
|:---|:---------|:-----|
| Prior | `/pro` | 何を学んだかの直感的振返り (前感情) |
| Likelihood | `/dox+` | 信念を詳細化して言語化 |
| Likelihood | `F:[×2]{/u+~(/noe*/dia)}` | 自問を2回、各回で直観×判定と振動させて深化 |
| Posterior | `~(/h*/k)` | 学びを動機×文脈の全体像に統合 |
| Posterior | `/pis` | 学びの定着度の確信度を測定 |
| Posterior | `/bye+` | 詳細ハンドオフで次セッションに記憶 |

## 使用例

```ccl
@learn                     # 標準学習永続化
@learn _ @vet              # 永続化した内容を検証
```
