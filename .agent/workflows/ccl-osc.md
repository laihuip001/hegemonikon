# /ccl-osc: 3項振動マクロ

> **CCL**: `@osc = /s~/dia~/noe`
> **用途**: Hub 内を往復して認識を深めたいとき

## 展開

- 戦略 ↔ 判定 ↔ 認識 を振動
- `/a~/h~/o~/s` の軽量版

## 使用例

```ccl

@osc+             # 詳細振動
```
