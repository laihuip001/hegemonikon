# /ccl-plan: 計画策定マクロ

> **CCL**: `@plan = /bou+_/chr_/s+~(/p*/k)_V:{/dia}`
> **用途**: 何かを始める前に計画を練りたいとき

## 展開

1. `/bou+` — 意志を詳細化
2. `_/s+~(/p*/k)` — 戦略を環境×文脈と振動融合
3. `_V:{/dia}` — 判定で検証ゲート

## 使用例

```ccl
@plan                      # 標準計画
@plan _ @build             # 計画後に構築
```
