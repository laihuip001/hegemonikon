# /pro 派生モード

> **定理**: H1 Propatheia (前感情・直感)
> **CCL演算子**: `/pro.{derivative}`

---

## appr — 接近傾向

> `/pro.appr` — 「引かれる」直感を精緻化

### 発動条件

- 何かに惹かれている
- 接近 (approach) 傾向の正体を探りたい

### 処理

```
入力: 接近的直感
  ↓
[1] 何に引かれているか (対象の同定)
[2] なぜ引かれるか (源泉の推定)
  ├─ 新奇性: 見たことがない → 好奇心
  ├─ 既知パターン: 過去の成功体験
  └─ 美: Hegemonikón 的美感
[3] 接近の妥当性 (X-HK1 バイアスチェック)
  ↓
出力: [接近対象, 源泉, バイアスリスク]
```

### CCL

```ccl
/pro.appr{toward: "新しいフレームワーク"} >> /dia{check: "novelty bias?"}
```

## avoi — 回避傾向

> `/pro.avoi` — 「引く」直感を精緻化

### 発動条件

- 何かを避けたい
- 回避 (avoidance) 傾向の根拠を探りたい

### 処理

```
入力: 回避的直感
  ↓
[1] 何を避けているか
[2] なぜ避けるか
  ├─ 恐怖: 失敗・損失の予感
  ├─ 嫌悪: 価値観の不一致
  └─ 疲労: エネルギー不足
[3] 回避の妥当性
  └─ 正当な回避 vs 怠惰な回避
  ↓
出力: [回避対象, 源泉, 妥当性]
```

### CCL

```ccl
/pro.avoi{away_from: "大規模リファクタ"} >> /pat{identify: "恐怖か怠惰か"}
```

## arre — 停止傾向

> `/pro.arre` — 「止まった」状態を診断

### 発動条件

- 接近も回避もしない
- フリーズ状態、判断停止

### 処理

```
入力: 停止状態
  ↓
[1] なぜ止まっているか
  ├─ 情報不足: 判断材料が足りない
  ├─ 矛盾: 接近と回避が拮抗
  └─ 過負荷: 選択肢が多すぎる
[2] 停止解除の方法
  ├─ 情報不足 → /sop で調査
  ├─ 矛盾 → /dia で判定
  └─ 過負荷 → /met でスケール変更
  ↓
出力: [停止原因, 解除方法]
```

### CCL

```ccl
/pro.arre{state: "frozen"} >> /dia.epo{then: "/sop or /met"}
```

## random — ランダム刺激

> `/pro.random` — 意図的にランダムな刺激で直感を揺さぶる

### 処理

- ランダムな概念/キーワードを提示
- それに対する直感反応を観察
- 予想外の接近/回避パターンを発見する

### CCL

```ccl
/pro.random >> /pat{observe: "何に反応したか"}
```

---

## forecast — 直感予測

> `/pro.forecast` — 将来の傾向を直感で予測

### 処理

- 「3ヶ月後、何に惹かれていると思うか」を問う
- 現在の直感パターンから未来を外挿
- Premortem (/bou.pre) との連携

### CCL

```ccl
/pro.forecast{horizon: "3_months"} >> /bou{long_term: true}
```
