# /rec ワークフロー (Recollect)

> **目的**: 長い会話で抜け落ちた記憶（ルール、スキル）を呼び起こす
> **頻度**: 5-10ラリーに1回、または「忘れてない？」と感じた時

---

## 発動条件

| トリガー | 説明 |
|----------|------|
| `/rec` | 明示的に記憶を呼び起こす |
| ユーザー「忘れてない？」 | 暗黙的なリマインド |

---

## 実行内容

```
1. 不変核ルールの確認
   - 3 Laws: Guard, Prove, Undo
   - 忖度禁止
   - 1作業1ターン原則

2. 現在のタスクに関連するスキルのリロード
   - 情報収集中 → M5 Peira
   - 計画中 → M4 Phronesis
   - 検証中 → M7 Dokime

3. 未完了タスクの確認
   - task.md を読み返す
```

---

## 出力形式

```markdown
[Hegemonikon] M8 Anamnēsis - Recollect

**不変核**:
- Guard: 大事なものには触らせない
- Prove: 動くと言う前にテストで示す
- Undo: 何をしても元に戻せる状態を保つ

**現在のタスク**: {task.md から}

**アクティブスキル**: {関連するM1-M8}

**直近の決定事項**: {会話から抽出}
```
