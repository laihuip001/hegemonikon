# /ccl-ground: 具体化マクロ

> **CCL**: `@ground = /pro_/tak-*/bou+{6w3h}~/p-_/ene-_/pis`
> **用途**: 抽象概念を具体的タスクに落とす
> **認知骨格**: Prior → Likelihood → Posterior

## 展開

| 相 | ステップ | 意味 |
|:---|:---------|:-----|
| Prior | `/pro` | 何を具体化したいかの直感 (前感情) |
| Likelihood | `/tak-` | 軽量タスク整理 |
| Likelihood | `*/bou+{6w3h}` | 意志に6W3Hモードで融合 |
| Likelihood | `~/p-` | 環境条件と振動 |
| Likelihood | `_/ene-` | 軽量実行へ接続 |
| Posterior | `/pis` | 具体化の十分さの確信度を測定 |

## 使用例

```ccl
@ground                    # 標準具体化
@ground _ @build           # 具体化後に構築
```
