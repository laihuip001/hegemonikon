# 位相反転演算子「`\`」(Antistrophē) Implementation Plan v2.0

> **Hegemonikón**: S2 Mekhanē + A2 Krisis
> **CCL署名**: `/s~\s_/u+`
> **Origin**: 2026-01-30 Creator との対話から創発

---

## 1. 設計思想

```
/WF = 正位相（ポジティブ視点）
\WF = 逆位相（ネガティブ視点）
```

> **「位相反転」は音響工学から借用した概念。波形を 180° 反転させると、合成時に打ち消し合うか、強め合う。**
> **思考も同様。正と逆を振動させることで、見落としが可視化される。**

---

## 2. 仕様

### 2.1 演算子定義

```yaml
symbol: "\"
name: Antistrophē (反転)
greek: ἀντιστροφή
type: 単項演算子（プレフィックス）
```

### 2.2 構文

| 構文 | 意味 |
|:-----|:-----|
| `\WF` | WF を逆位相で実行 |
| `\WF+` | 逆位相で深化 |
| `/WF~\WF` | 正逆を振動（両面を見る） |
| `/WF~\WF_/u+` | 正逆振動 → 私が統合 |

### 2.3 等価表記

```
\s  ≡  /s:1  ≡  /s@neg
/s  ≡  /s:0  ≡  /s@pos  (デフォルト)
```

### 2.4 強制ルール（反転実行時）

| ルール | 説明 |
|:-------|:-----|
| **肯定禁止** | 「良い点」「利点」への言及禁止 |
| **buttering 禁止** | 「〜だが」「しかし良い面も」禁止 |
| **否定先行** | 必ず批判から開始 |
| **3 点必須** | 最低 3 つの批判点を出力 |
