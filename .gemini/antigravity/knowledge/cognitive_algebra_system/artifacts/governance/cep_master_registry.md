# CEP Master Registry & Framework Governance

## 1. The CEP Process (CEP-000)

CCL evolution follows the lightweight **CEP (CCL Enhancement Proposal)** process.

### Principles

- **Freeze Threshold**: Maximum 15 active symbols.
- **Symmetry**: Balanced pair logic (深化/縮約, 上昇/根源).
- **Life Cycle**: Draft → Proposed → Accepted → Final.

## 2. Symbolic Extensions (CEP-001)

Integration of **Active Inference / FEP** concepts into the symbolic vocabulary.

| Symbol | Name | Calculus / FEP Equivalent |
|:---|:---|:---|
| `'` | Prime | Rate of change (dε/dt / 変化率) |
| `√` | Root | 基底抽出 / Root Cause (Inversion of ^) |
| `→ (>>)` | Limit | Convergence / Prediction Error Minimization |
| `∫` | Integral | History Integration / Accumulation (履歴統合) |
| `Σ` | Sum | Aggregation of multiple results |
| `∏` | Product | Fusion of multiple elements |
| `∂` | Partial | Change relative to specific dimension |
| `log` | Log | Evaluation of Information Gain / Entropy |
| `E[]` | Expectation | Predictive output (平均的成果) |
| `V[]` | Variance | Uncertainty / Entropy (不確実性) |

### Formalized Shorthands

- `lim[cond]{expr}` is the formal form of `expr >> cond`.
- `I: cond {expr}` is the formal form of `cond ? expr`.
