## 3. Operational Layer τ (τ層)

Hegemonikón における **τ層 (Operational Layer)** は、認知プロセスそのものではなく、システムのライフサイクル、環境、およびメタ運用を管理するためのワークフロー群。

### 3.1 τ層ワークフロー一覧

| コマンド | 役割 | 認知的同期 |
|:---|:---|:---|
| `/boot` | セッション初期化・文脈復元 | O1, H4 準拠 |
| `/bye` | セッション終了・文脈の保存 | H4 準拠 |
| `/dev` | 開発プロトコル参照 | P4 準拠 |
| `/sop` | 検索・探索プロトコル参照 | K4 準拠 |
| `/eat` | 外部コンテンツの同化 | S2, A2 準拠 |

### 3.2 Cognitive Algebra の適用

- **`-` (縮約)**: 運用負荷の最小化（例: `/boot-` 高速ブート）。
- **`+` (深化)**: 詳細な状況確認（例: `/dev+` モジュール全文展開）。
- **`*` (メタ)**: 運用プロトコルそのものの妥当性検証。
