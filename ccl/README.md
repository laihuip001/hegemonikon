# CCL (Cognitive Control Language)

> **Hegemonikón の顔（Interface）** — 認知プロセスを代数的に記述する言語

## はじめに

CCL は Hegemonikón を「呼び出す」言語です。Python における構文 (Syntax) に相当し、
FEP (公理) と定理群 (builtins) を操作するためのインタフェースを提供します。

## クイックスタート

| したいこと | CCL 式 | 説明 |
|:-----------|:-------|:-----|
| 深く考える | `/noe+` | 認識を詳細展開 |
| 意志を明確にする | `/bou+` | 意志を深掘り |
| 設計→実行 | `/s+_/ene` | 設計後に実行 |
| 両面を見る | `/noe~\noe` | 正逆の往復 |
| 融合+メタ | `/noe*^dia` | 統合して過程も表示 |
| 射で変換 | `/noe >> /dia` | 認識を判定に構造変換 |
| 収束ループ | `C:{/dia+_/ene+}` | 修正サイクルを回す |
| 変化率を見る | `/bou'` | 意志がどう変わったか |
| 不確実性を測る | `V[/noe]` | 予測誤差の分散 |
| 履歴を統合 | `∫/dox` | 信念の累積 |

## 演算子一覧

→ 詳細は [operators.md](operators.md) を参照

### 単項演算子

| 記号 | 名称 | 作用 |
|:-----|:-----|:-----|
| `+` | 深化 | 詳細化、深掘り |
| `-` | 縮約 | 要約、最小出力 |
| `^` | 上昇 | メタ層へシフト |
| `√` | 下降 | 具体層へシフト |
| `\` | 反転 | 視点を逆転 |
| `!` | 全展開 | 全派生を並列展開 |

### FEP 演算子

| 記号 | 名称 | 作用 | 例 |
|:-----|:-----|:-----|:---|
| `'` | 微分 | 変化率 (dε/dt) | `/bou'` |
| `∂` | 偏微分 | 特定次元の変化率 | `∂s1/bou` |
| `∫` | 積分 | 履歴統合・累積 | `∫/dox` |
| `V[]` | 分散 | 予測誤差の不確実性 | `V[/noe]` |
| `lim` | 収束 | 極限・最小化の終着点 | `lim[ε=0]{/s~dia}` |

### 二項演算子

| 記号 | 名称 | 作用 |
|:-----|:-----|:-----|
| `*` | 融合 | 精度加重された統合 ⟨A,B⟩_π |
| `%` | 展開 | 全次元の組み合わせを保持 A ⊗ B |
| `~` | 振動 | 複数を往復して探索 |
| `~*` | 収束振動 | 融合するまで往復 |
| `~!` | 発散振動 | 展開的に往復 |
| `_` | シーケンス | Aの後にBを実行 |
| `>>` | 射 | Aを構造的にBに変換 |
| `>*` | 変容 | AをBの視点で変容 |

### 記号構文 (CPL デコレータ圧縮)

| 記号 | 意味 |
|:-----|:-----|
| `C:{X}` | 収束ループ |
| `R:{X}` | 累積融合 |
| `V:{X}` | 検証ゲート |

### 制御構文 (CPL)

| 構文 | 意味 |
|:-----|:-----|
| `F:[×N]{X}` | N回反復 |
| `F:[A,B]{X}` | 各要素に適用 |
| `I:[cond]{X}` | 条件分岐 |
| `E:{X}` | ELSE |
| `W:[cond]{X}` | ループ |
| `L:[x]{X}` | Lambda |

## Python との対応

| CCL 概念 | Python 対応 |
|:---------|:------------|
| CCL 式 | 式 (Expression) |
| 演算子 | `operator` モジュール |
| WF | 関数定義 |
| マクロ (@) | デコレータ (@) |
| 定理 | builtins |
| C:/R:/M:/V: | functools/itertools |
| F:/I:/E:/W: | for/if/else/while |
