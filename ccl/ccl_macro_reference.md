## 構文体系

### CCL 演算子（ネイティブ）

| 演算子 | 意味 |
|:-------|:-----|
| `_` | チェーン（直列化） |
| `~` | 振動（行き来） |
| `*` | 融合 |
| `*^` | メタ融合 |
| `+` | 深化 |
| `-` | 簡略化 |
| `{params}` | パラメータ |
| `(group)` | グループ |

### 制御構文（CPL）

| 構文 | 意味 | Python 対応 |
|:-----|:-----|:-----------|
| `F:[×N]{X}` | N回反復 | `for _ in range(N)` |
| `F:[a,b]{X}` | 各要素に適用 | `for x in [a,b]` |
| `I:[cond]{X}` | 条件分岐 | `if cond:` |
| `E:{X}` | その他 | `else:` |
| `L:[x]{X}` | Lambda | `lambda x: X` |

### 記号構文（CPL デコレータ圧縮）

| 記号 | 意味 | 元の名称 |
|:-----|:-----|:---------|
| `C:{X}` | 収束ループ | @cycle |
| `R:{X}` | 累積融合 | @reduce |
| ~~`M:{X}`~~ | ~~記憶~~ (廃止) | ~~@memoize~~ |
| `V:{X}` | 検証ゲート | @validate |

## MECE 分類（複数軸）

### 軸1: Flow × Value

```
              E (認識的)          P (実用的)
           ┌──────────────┬──────────────┐
I (推論)   │ @dig @nous   │ @plan @tak   │
           │ @osc         │ @ground      │
           ├──────────────┼──────────────┤
A (行為)   │ @vet @learn  │ @fix @build  │
           │ @proof       │              │
           └──────────────┴──────────────┘
```

### 軸2: FEP 予測誤差処理

| 検出 | 修正 | 防止 |
|:-----|:-----|:-----|
| @dig, @vet, @osc | @fix, @build | @plan, @tak |
| @nous | @ground | @learn, @proof |
