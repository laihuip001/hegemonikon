# CCL 応用例

## 複合式

```ccl
/bou~zet+       # 振動しつつ各ステップを詳細に
/noe*dia^       # 融合した後、メタ分析
/zet~bou/       # 振動した後、具体的行動に落とす
/o*h*s          # 三重融合: 認知×動機×設計
```

## 修飾子付き

```ccl
/bou+s1@4       # 人生レベル（実存的視点）で意志を問う
/noe+k2@1       # 月単位の時間展望で現状を認識
/dia+a3         # 原則レベルで判定
```

## 反転 (Antistrophē)

```ccl
\s+             # 設計を逆位相で批判
/noe~\noe       # 正逆から認識
/bou~\bou_/u+   # 両面を見て AI が統合
```

## ターゲット指定

```ccl
[CCL]/mek+      # CCL プログラムを生成
[WF]/dia-       # WF を縮約監査
[KI]/zet+       # 知識項目を深堀り探索
```

## パイプライン

```ccl
/bou+_/s+_/ene  # 意志明確化 → 詳細設計 → 実行
/boot_/bou      # ブート後に意志確認 (Cognitive Warm-up)
```

## 爆発と収束

```ccl
/bou!~/zet-     # 全意志を広げて、核心の問いに絞る
```
