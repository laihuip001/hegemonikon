# CCL 基本例

## 単項演算子

```ccl
/noe+     # 深い認識を詳細に
/noe-     # 深い認識の要点のみ
/noe^     # 「なぜこの問いを問うのか」
/noe/     # 「結局、今日何をする？」
```

## 二項演算子

```ccl
/bou*zet  # 意志と探求を融合 →「私が知りたいことは何か」
/bou~zet  # 意志と探求を往復 → 欲求と問いの対話
/s+_/ene  # 設計してから実行
```

## 基本パターン

| ユースケース | CCL 式 |
|:-------------|:-------|
| 深く考えたい | `/noe+` |
| 何をすべきか知りたい | `/bou+` |
| 設計→実行の流れ | `/s+_/ene` |
| 批判的に検証 | `/dia+` |
| 軽く方向性だけ | `/bou-` |
