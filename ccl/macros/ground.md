# @ground マクロ

> **定義**: `/tak/`

## 目的

抽象的概念を具体的な現実（接地/Grounding）へと着地させる。

## 構文

```ccl
@ground <抽象概念>
```

## 実行プロセス

1. 抽象的入力を受け取る
2. 6W3H で構造化
3. 期限・依存・工数を明示
4. 実行可能なタスクとして出力

## 使用例

```ccl
@ground "AIを賢くしたい"     # 抽象的な願望を具体化
@ground /bou+ の出力         # 意志の結果を接地
```

## 接地の基準

- **What**: 何を
- **Why**: なぜ
- **Who**: 誰が
- **When**: いつ
- **Where**: どこで
- **Which**: どれを
- **How**: どうやって
- **How many**: いくつ
- **How much**: いくら
