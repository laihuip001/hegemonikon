## 2. 二項演算子 (3個)

| 記号 | 名称 | 作用 | タスク構造 |
|:-----|:-----|:-----|:----------|
| `*` | 融合 | 複数を統合して1出力 | シングルタスク |
| `~` | 振動 | 複数を往復して探索 | マルチタスク |
| `_` | シーケンス | Aの後にBを実行 | 逐次処理 |

### 区別

```
*  = 「両方踏まえて一つの答えをくれ」 → 結果重視
~  = 「両方の対話を見せてくれ」     → プロセス重視
_  = 「Aの次にBをやってくれ」       → 手続き
```

## 3. 特殊パターン

### 3.1 融合メタ表示 `*^`

```
/noe*dia    → 融合結果のみ
/noe*dia^   → 融合結果 + どう融合したかのメタ解説
```

### 3.2 定理修飾子

| 記号 | 対象 | 作用 |
|:-----|:-----|:-----|
| `+s1` | スコープ | Micro → Macro 拡大 |
| `+k2` | 時間軸 | 今日 → 長期展望 |
| `+p1` | 領域 | 局所 → 全体コンテキスト |
| `+a3` | 原則 | 具体 → 普遍的原則 |

### 3.3 レベルパラメータ `:N`

| 表記 | 意味 | スケール |
|:-----|:-----|:---------|
| `+s1:1` | 1段階拡大 | 月/中域 |
| `+s1:4` | 4段階拡大 | 人生/永劫 |

## 4. 構文規則 (v1.2)

1. **評価順序**: 左から右へ（思考の順序）
2. **先頭単項**: 式全体にかかる
3. **末尾単項**: 直前のWFにかかる
4. **括弧**: 複雑な式でスコープを明示
