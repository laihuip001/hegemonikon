## 背景

Hegemonikón に新しいレイヤー「Synteleia（認知アンサンブル）」を追加した。
6つのエージェントが並列で認知処理を行い、結果を統合するアーキテクチャ。

### 設計概要

```
6 Agents (O/H/A/S/P/K) × 3 Phases (処理/出力/監査) = 18 セル
```

各Agent は Hegemonikón の6カテゴリから演繹的に導出:
- O-Agent: 本質洞察
- H-Agent: 動機評価
- A-Agent: 精度判定
- S-Agent: 構造設計
- P-Agent: 境界画定
- K-Agent: 時宜判断
