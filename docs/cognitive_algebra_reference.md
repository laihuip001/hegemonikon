## 二項演算子 (定理間複合)

| 演算子 | 意味 | 読み方 | 例 | ユースケース |
|:-------|:-----|:-------|:---|:-------------|
| `~` | **Oscillation** | 「と」を往復 | `/bou~zet` | 意志と探求を交互に深める |
| `×` | **Fusion** | 「かける」 | `/noe×dia` | 認識しながら同時に判定 |
| `&` | **Parallel** | 「and」 | `/zet&sop` | 問いを立てながら調査 |

### 複合例

| 式 | 意味 | いつ使う |
|:---|:-----|:---------|
| `/bou~zet` | 意志↔探求を振動 | やりたいことと問いを交互に明確化 |
| `/bou~zet~noe` | 意志↔探求↔認識を三項振動 | 深い方向性決定時 |
| `/noe×dia` | 認識⊗判定を融合 | 分析しながらレビュー |
| `/zet&sop` | 探求∧調査を並列 | 問いを立てながら情報収集 |

## Hub = Series 内部4項振動 (v3.0 実装済み)

> `/o` = `/noe~bou~zet~ene` — Hub は「ただの選択メニュー」ではなく「4項振動」

| Hub | 展開形 | 意味 |
|:----|:-------|:-----|
| `/o` | `/noe~bou~zet~ene` | 認識 ↔ 意志 ↔ 探求 ↔ 行為 |
| `/s` | `/met~mek~sta~pra` | スケール ↔ 方法 ↔ 基準 ↔ 実践 |
| `/h` | `/pro~pis~ore~dox` | 傾向 ↔ 確信 ↔ 欲求 ↔ 信念 |
| `/p` | `/kho~hod~tro~tek` | 空間 ↔ 経路 ↔ 軌道 ↔ 技法 |
| `/k` | `/euk~chr~tel~sop` | 好機 ↔ 時間 ↔ 目的 ↔ 知恵 |
| `/a` | `/pat~dia~gno~epi` | 感情 ↔ 判定 ↔ 原則 ↔ 知識 |

### Hub 演算子

| 式 | 意味 |
|:---|:-----|
| `/o` | O-series 4項振動 (内部振動) |
| `/o+` | 全4定理を詳細モードで順次実行 |
| `/o-` | 全4定理を縮約モードで順次実行 |
| `/o*` | 「なぜ O-series か」を問う |
| `/o~k` | O-series ↔ K-series 間振動 (Series間振動) |

### 設計決定

- `~` は**常に二項演算子** (`/o~k` = Series間振動)
- Hub 単体 (`/o`) = 4項振動 (内部振動)
- 個別定理へは直接アクセス (`/noe`, `/bou`...)
