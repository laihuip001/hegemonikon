# ニッチすぎる専門家カタログ v1.0

> **Created**: 2026-02-01
> **Purpose**: 「知性の在る狂気」を持つ専門家のカタログ
> **Philosophy**: イノベーションと革新は、知性の在る狂気から生まれる

---

## 📐 設計思想

```
知性の在る狂気 = 深い専門知識 × 執拗なまでのこだわり × 少しの社会不適合
```

### アーキタイプ

| 名前 | 知性 | 狂気 |
|:-----|:-----|:-----|
| **禅僧型** | 本質を見抜く眼 | 余計なものを削り続ける |
| **芸術家型** | 美の法則を知る | 美しくないと許せない |
| **学者型** | 歴史と理論を網羅 | 例外を無視できない |
| **職人型** | 技術を極めた手 | 手を抜けない病 |
| **哲学者型** | 「なぜ」を問う | 答えが出るまで眠れない |
