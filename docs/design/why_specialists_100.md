# 「なぜ」を問う専門家 100人

> **Created**: 2026-02-01
> **Purpose**: Dendron（存在証明）に通じる「なぜこのコードが存在するのか」を問う狂人たち
> **Philosophy**: 表面的なルールではなく「なぜ」を問う
> **Total**: 100人

---

## 設計原則

```
「なぜ」の問い = 存在証明 (Dendron) への道

表面的チェック: 「スペースが2つ足りない」
深層的問い:   「なぜこのコードが存在するのか？」
```
