# FEPにおける知覚と認知の関係性

FEPでは、知覚も認知も「生成モデルに基づくベイズ推論」という同一の計算原理で記述され、階層の違い（時間スケール・抽象度）として統一的に扱われる。知覚はより低次レベルでの原因推定、認知はその上位レベルでの抽象的な原因や方略の推定として、同じ自由エネルギー最小化過程の別側面とみなされる。[^1_1][^1_2][^1_3]

## FEPの基本枠組み

- 自由エネルギー原理では、生体や認知エージェントは「感覚入力の驚き（surprisal）の上限である変分自由エネルギーを最小化するように内部状態と行動を変化させ続ける」とされる。[^1_3][^1_4][^1_1]
- このとき世界の隠れた原因を記述する**生成モデル**と、それに対する事後分布（信念）が中心的役割を果たし、知覚・学習・行為選択はすべてこの生成モデルに関するベイズ推論として表現される。[^1_5][^1_1][^1_3]


## 知覚：低次階層でのモデル反転

- 知覚は「現在の感覚入力の原因を最もよく説明するように、内部信念（事後分布）を更新する過程」として、生成モデルの“反転”＝変分自由エネルギー最小化と同一視される。[^1_2][^1_6][^1_3]
- 階層的予測符号化モデルでは、高次レベルが下位レベルの状態を予測し、下位レベルからの予測誤差（prediction error）が上向きに伝搬し、それを最小化する方向に活動・接続が更新されることで知覚推論が実現すると説明される。[^1_7][^1_8][^1_2]


## 認知：高次階層でのモデル操作

- 認知（思考・推論・意思決定など）は、同じ自由エネルギー最小化を「より高次・抽象的なレベル」で行う過程として理解される。具体的には、長期的・文脈的な隠れ状態（目標、規則、他者の意図など）に対する信念の更新・保持・操作が認知に相当する。[^1_9][^1_10][^1_1]
- 例えばWCSTのような課題の計算モデルでは、FEPに基づく階層RNNが「カテゴリー（ルール）」という高次の隠れ状態を推定しつつカードの特徴との予測誤差を最小化することで、柔軟な概念シフト＝認知的柔軟性を再現している。[^1_11]


## 階層構造と知覚‐認知の連続性

- FEP／予測符号化では、皮質は階層的に組織され、下位では感覚特徴・運動パラメータ、上位では物体・場面・目標・自己や他者モデルなど、より抽象的・長時間スケールの変数が表現されると想定される。[^1_12][^1_1][^1_2]
- この階層において、知覚と呼ばれるプロセスは主に下位〜中位レベルの信念更新、認知と呼ばれるプロセスは中位〜上位レベルの信念更新として連続的に位置づけられ、両者の間に原理的な断絶はなく「同じ機構の異なる時間・抽象度スケール」という関係になる。[^1_1][^1_5][^1_3]


## 行為・アクティブ・インファレンスとの関係

- FEPは、自由エネルギー最小化を「内部状態の更新（知覚・認知）」と「外界を変える行為（アクション）」の両方で達成できるとする。前者は予測誤差を説明するように信念を変える、後者は予測に合うように感覚入力そのものを変えることに相当する。[^1_4][^1_5][^1_3]
- アクティブ・インファレンスでは、方策レベルの認知（将来の結果に関する信念や好み）も生成モデルの一部として扱われ、将来の自由エネルギー／期待サプライズを最小化する行為系列の選択として意思決定が定式化されるため、認知プロセスは知覚・行為と一体の推論ダイナミクスとして統合される。[^1_13][^1_3][^1_12]


## まとめ的整理（関係性の一文表現）

- 関係を一文で圧縮すると、「FEPでは、知覚＝感覚入力の即時的な原因推定、認知＝より抽象的・長期的な原因や方策の推定であり、いずれも生成モデルに基づく自由エネルギー最小化（ベイズ推論）の階層的レベル差として統一的に記述される」と表現できる。[^1_2][^1_5][^1_1]

# FEPに基づき”見つける/考える/動く/振り返る”という４つの概念を階層構造や関係性で説明してみて

FEPでは、「見つける」「考える」「動く」「振り返る」は、すべて変分自由エネルギー最小化という統一原理の下で、階層的かつ循環的に組織された異なる側面として理解される。知覚‐推論‐行為‐学習のサイクルは、時間スケールと抽象度によって区別される同一プロセスの反復として統合される。[^2_1][^2_2][^2_3][^2_4]

## 見つける：知覚推論と探索的情報獲得

「見つける」は、FEPにおける**知覚推論（perceptual inference）**と**探索（exploration）**に対応し、現在の感覚入力から隠れた原因を推定するとともに、不確実性を減らすために情報価値の高い状態をサンプリングする過程である。[^2_2][^2_5][^2_4]

- 低次階層では、予測符号化により予測誤差を最小化する形で外界状態の事後分布を更新し、知覚的「発見」が生じる。[^2_6][^2_7]
- 期待自由エネルギー（EFE）の内因的価値（intrinsic value）成分は「その行為を選べば隠れた原因について何が分かるか」を定量化し、エピステミック駆動（epistemic drive）として探索を動機づける。曖昧性を減らす観察が価値をもつため、未知の領域を「見つけに行く」行動が自然に生まれる。[^2_5][^2_8][^2_4]


## 考える：方策推論と計画

「考える」は、FEPの**方策推論（planning as inference）**に相当し、将来の状態・行為系列に対する信念を形成し評価する高次の推論プロセスである。[^2_4][^2_1][^2_2]

- アクティブ・インファレンスでは、可能な行為系列（方策）それぞれについて期待自由エネルギーを計算し、その最小化が「計画」として実装される。この過程で、将来の状態を予測し、目標との整合性と情報獲得の両方を考慮した推論が展開される。[^2_1][^2_5][^2_4]
- 「高度な推論（sophisticated inference）」の枠組みでは、「もしあの行動を取れば自分はどう信じるか」という再帰的信念更新を行い、より深い意思決定ツリー探索が可能になる。これは方策レベル・目標レベルの認知に相当し、階層の上位で長い時間スケールにわたる「考える」を担う。[^2_9][^2_10][^2_4]


## 動く：行為による自由エネルギー最小化

「動く」は、FEPの**行為（action）**すなわち**アクティブ・インファレンス**そのものであり、信念を世界に合わせるのではなく世界を信念に合わせる方向で自由エネルギーを最小化する過程である。[^2_11][^2_4][^2_6]

- 期待自由エネルギー最小化は、リスク最小化（予測される状態を好ましい状態に近づける）と曖昧性最小化（情報を得られる観察を採取する）の統合であり、行為選択は「目標達成（exploitation）」と「情報獲得（exploration）」を原理的に両立させる。[^2_2][^2_5][^2_4]
- 行為は知覚・推論と分離されず、「予測誤差を減らすために感覚入力を変える」ことで同じ推論ループの出力として位置づけられる。筋骨格系の制御も予測符号化による誤差最小化として説明され、知覚‐行為は連続的な自由エネルギー最小化ダイナミクスとなる。[^2_12][^2_4][^2_6]


## 振り返る：生成モデル更新と学習

「振り返る」は、経験を通じた**生成モデルの更新（learning）**および**メタ認知的なモデル評価・選択**に対応し、より長期的な時間スケールで自由エネルギーを低減させる過程である。[^2_13][^2_14][^2_15]

- 短期的には知覚・行為が変分自由エネルギーを最小化するのに対し、長期的には生成モデルのパラメータ（シナプス結合強度など）や構造自体を更新することで、継続的に驚きを低減する。これは経験の「振り返り」を通じた学習として現れる。[^2_15][^2_13]
- 自由エネルギー最小化は、「現在の軌道がモデルのエネルギー勾配を下る」という形で実装され、実験的には神経回路のシナプス可塑性と神経応答の変化が自由エネルギー原理で予測可能であることが示されている。これは試行を繰り返す中での内省的「振り返り」と更新プロセスに相当する。[^2_13]
- さらに、精度（precision）パラメータの調整やモデル選択のレベルでは、自身の信念の確信度を推定するメタ認知的機能が自然発生し、これは「自分がどう推論しているか」を振り返る高次の自己モニタリングとして解釈できる。[^2_16][^2_14]


## 階層構造と循環的統合

これら4つの概念は、時間スケールと階層の違いとして以下のように整理される。[^2_10][^2_9][^2_2]


| 概念 | FEP機能 | 時間スケール | 階層レベル |
| :-- | :-- | :-- | :-- |
| **見つける** | 知覚推論・探索（エピステミック価値） | 短期（数百ms～数秒） | 低次（感覚皮質～中位） |
| **考える** | 方策推論・計画（期待自由エネルギー評価） | 中期（秒～分） | 中位～高次（前頭前野など） |
| **動く** | 行為選択・実行（リスク＋曖昧性最小化） | 短～中期（数百ms～秒） | 運動階層全体 |
| **振り返る** | 生成モデル学習・メタ認知（モデル更新） | 長期（試行間～生涯） | 最高次（モデル選択・精度調整） |

実際には、これらは循環的に相互作用する。探索して「見つけた」情報は「考える」推論を更新し、「動く」行為選択に反映され、その結果を「振り返り」学習することで生成モデルが洗練され、再び新たな「見つける」推論の精度が向上する。[^2_9][^2_4][^2_2]

# 2×2の構造、又はほかの概念を補填して2×2×2、又は2×2×2×2の対比構造に分類できる？

FEPの枠組みでは、複数の対比軸を用いて「見つける・考える・動く・振り返る」を体系的に拡張できます。特に**推論vs行為**、**エピステミックvsプラグマティック**、**短期vs長期**、**低次vs高次階層**という4軸が理論的に重要です。[^3_1][^3_2][^3_3][^3_4]

## 2×2構造：基本的な対比

最も基本的な分類は、**情報の流れ（推論 vs 行為）**と**価値の種類（エピステミック vs プラグマティック）**の2軸です。[^3_3][^3_5][^3_1]


|  | **エピステミック価値**<br>（不確実性削減・情報獲得） | **プラグマティック価値**<br>（目標達成・報酬最大化） |
| :-- | :-- | :-- |
| **推論**<br>（内向き・信念更新） | **見つける**<br>知覚推論、仮説生成 | **判断する**<br>目標志向推論、評価 |
| **行為**<br>（外向き・世界介入） | **試す**<br>探索行動、実験 | **動く**<br>搾取行動、目標実行 |

期待自由エネルギー（EFE）は、エピステミック項（情報獲得による不確実性削減）とプラグマティック項（好ましい結果への到達）の和として分解され、探索-利用ジレンマを統一的に解決します。[^3_2][^3_1][^3_3]

## 2×2×2構造：時間スケールの追加

上記に**時間スケール（短期 vs 長期）**軸を加えると、8つのプロセスに分類できます。[^3_6][^3_7][^3_4]


| 時間 | 推論/行為 | エピステミック | プラグマティック |
| :-- | :-- | :-- | :-- |
| **短期**<br>（試行内・秒単位） | 推論 | **見つける**<br>即時知覚推論 | **判断する**<br>即時評価・選択 |
|  | 行為 | **試す**<br>探索的サンプリング | **動く**<br>目標志向行為 |
| **長期**<br>（試行間・分～生涯） | 推論 | **仮説を立てる**<br>モデル構造探索 | **考える**<br>戦略的計画・方策評価 |
|  | 行為 | **実験する**<br>系統的探索設計 | **振り返る**<br>モデル更新・学習 |

短期プロセスは現在の信念と方策に基づく変分自由エネルギー最小化、長期プロセスは生成モデルのパラメータや構造自体の更新を通じた自由エネルギー低減に対応します。[^3_7][^3_4][^3_8]

## 2×2×2×2構造：階層レベルの統合

さらに**階層レベル（低次 vs 高次）**を加えると、16の認知-行為モードが区別されます。[^3_9][^3_4][^3_10][^3_11]

### 低次階層（感覚運動レベル・ミリ秒～秒）

| 時間/流れ | エピステミック | プラグマティック |
| :-- | :-- | :-- |
| **短期×推論** | **見つける<sub>感覚</sub>**<br>感覚特徴検出、予測誤差処理 | **判断する<sub>運動</sub>**<br>運動パラメータ選択 |
| **短期×行為** | **試す<sub>サッケード</sub>**<br>眼球運動による情報獲得 | **動く<sub>反射</sub>**<br>定型的運動実行 |
| **長期×推論** | **仮説立て<sub>感覚</sub>**<br>感覚統計の再推定 | **考える<sub>運動</sub>**<br>運動シーケンス計画 |
| **長期×行為** | **実験<sub>探索</sub>**<br>感覚空間の系統走査 | **振り返る<sub>運動</sub>**<br>運動学習・技能獲得 |

### 高次階層（概念・目標レベル・秒～年）

| 時間/流れ | エピステミック | プラグマティック |
| :-- | :-- | :-- |
| **短期×推論** | **見つける<sub>概念</sub>**<br>文脈認識、意味推論 | **判断する<sub>目標</sub>**<br>目標評価・優先順位づけ |
| **短期×行為** | **試す<sub>質問</sub>**<br>能動的質問・確認行動 | **動く<sub>決定</sub>**<br>意思決定・コミット |
| **長期×推論** | **仮説立て<sub>理論</sub>**<br>因果モデル構築 | **考える<sub>戦略</sub>**<br>長期計画・人生設計 |
| **長期×行為** | **実験<sub>科学</sub>**<br>実験設計・介入研究 | **振り返る<sub>メタ</sub>**<br>メタ学習・世界観更新 |

階層構造は時間スケールの入れ子に対応し、低次レベルは高次の信念に条件づけられつつ、高次レベルは低次からの予測誤差により更新されます。[^3_4][^3_11]

## 統合的理解：期待自由エネルギーの分解

これら16モードは、期待自由エネルギーの異なる時空間スケールでの最小化として統一されます。[^3_1][^3_2][^3_7]

$$
G(\pi) = \underbrace{\mathbb{E}_Q[\ln Q(s|\pi) - \ln P(o,s)]}_{\text{プラグマティック項}} + \underbrace{\mathbb{E}_Q[H[P(o|s)]]}_{\text{エピステミック項}}
$$

- **左辺（プラグマティック）**: 予測状態が好ましい観察をもたらす度合い → 「判断・動く・考える・振り返る」
- **右辺（エピステミック）**: 予測観察が状態について情報を与える度合い → 「見つける・試す・仮説・実験」

行為は期待自由エネルギーを最小化する方策を選択し、推論は観察を説明する隠れ状態を推定することで、両者が協調して全体の自由エネルギーを低減させます。[^3_5][^3_2][^3_1]

階層と時間スケールの違いにより、同じ原理が「反射的サッケード」から「科学的実験設計」まで、あらゆる認知-行為プロセスを生成します。[^3_11][^3_9][^3_4]
