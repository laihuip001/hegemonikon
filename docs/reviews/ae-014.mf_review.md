# 比喩一貫性の詩人 レビュー

## 対象ファイル
`mekhane/api/server.py`

## 判定
発言（要改善）

## 発見事項
- **(Medium) 比喩混在**:
  - `create_app` (Creation/Factory)
  - `_register_routers` (Bureaucracy/Registry)
  - `_cleanup_stale_socket` (Janitorial/Maintenance)
  - 改善案: 「構築 (Construction)」または「機械 (Machine)」の比喩で統一すべき。`register` (登録) よりも `mount` (実装/据え付け) が適切。
- **(Low) 動詞不統一**:
  - `cleanup` は名詞（大掃除）であり、動詞句 `clean_up` と混同されている。動詞として使うなら `clean` や `remove` が適切。
- **(Low) 抽象度不統一**:
  - `_register_routers` (高レベルなモジュール結合)
  - `_cleanup_stale_socket` (低レベルなファイル操作)

## 重大度
Medium
