# 比喩一貫性の詩人 レビュー

## 対象ファイル
`mekhane/anamnesis/gnosis_chat.py`

## 判定
発言（要改善）

## 発見事項
- (Medium) **比喩の混在**: `Gnosis` (霊的知) や `Anamnesis` (想起) という神秘的・精神的なメタファーを持つモジュール内で、`KnowledgeIndexer` (索引作成機) や `_build_context` (構築) といった「工場/建設」のメタファーが混在しています。`discover_knowledge_files` (発見/庭園) もあり、世界観が統一されていません。
- (Low) **動詞の不統一**: データの取得に関して、メソッド名は `_retrieve` ですが、内部変数には `fetch_k` が使われています。また外部呼び出しには `_get_embedder` が見られ、`retrieve/fetch/get` が混在しています。
- (Low) **抽象度の不統一**: `KnowledgeIndexer` は単なる関数群（static methods）の集合体ですが、状態を持つ `GnosisChat` (主役/Manager) と同列のクラスとして定義されています。
