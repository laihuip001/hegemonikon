# 比喩一貫性の詩人 レビュー

## 対象ファイル
`hermeneus/src/dispatch.py`

## 判定
発言（要改善）

## 発見事項
- **比喩混在 (Medium)**: モジュール全体は「物流・工場（Dispatch, Workflow, Template）」の比喩で構成されていますが、`format_ast_tree` 関数および戻り値のキー `tree` に「庭園（Tree）」の比喩が混入しています。技術用語としての AST (Abstract Syntax Tree) は許容されますが、変数名や関数名での `Tree` の強調は世界観を損ないます。
    - 提案: `format_ast_structure` または `visualize_ast` に変更し、キーも `structure` とする。
- **命名規則不統一 (Low)**: 関数名において `format_ast_tree` (Verb-Noun-Noun) と `extract_workflows` (Verb-Noun) で粒度が異なります。
    - 提案: `format_structure` に変更することで `extract_workflows` と動詞+目的語の形式を統一する。

## 重大度
Medium
