# 比喩一貫性の詩人 レビュー

## 対象ファイル
`mekhane/mcp/sympatheia_mcp_server.py`

## 判定
発言（要改善）

## 発見事項
- **比喩体系の混在 (Metaphor Mixing)** (Medium)
  - 「Sympatheia（自律神経系）」という生物学的比喩（WBC, Feedback/Homeostasis）の中に、物理学（Attractor）、ネットワーク（Route）、一般IT（Status）の用語が混入しています。
  - 庭園（バイオ）に工場（メカ）の看板が立っている状態です。特に `sympatheia_status` は `sympatheia_vitals`（バイタル）であるべきです。

- **抽象度の不統一 (Abstraction Level Mismatch)** (Low)
  - `sympatheia_wbc` 等は `sym` モジュールへの委譲（Manager role）に徹していますが、`sympatheia_status` は内部データ構造への直接アクセス（Worker role）を行っています。
  - 修正提案: `sym` モジュールに `get_system_vitals()` を実装し、MCPサーバーからはそれを呼ぶだけにすべきです。

- **動詞の不統一 (Verb Inconsistency)** (Low)
  - `_get_sympatheia` (Get) と、内部で使用している `_read_json` (Read), `_load_config` (Load) の使い分けに意図が見えません。情報の取得は統一された動詞体系（例: Retrieve/Fetch）で行うべきです。

## 重大度
Medium
