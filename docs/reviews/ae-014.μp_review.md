# 比喩一貫性の詩人 レビュー

## 対象ファイル
`mekhane/pks/pks_cli.py`

## 判定
発言（要改善）

## 発見事項
- (Metaphor Mixing: Medium) 同一モジュール内で「鉱業（Nugget）」と「サイバネティクス（Push/Feedback）」、「物理学（Attractor）」、「神秘主義（Gnōsis）」、「機械（Engine）」、「Web（Backlinks）」の比喩が混在している。特に `nuggets = engine.proactive_push(...)` は「掘り出すもの」を「プッシュする」という不自然な組み合わせになっている。
- (Naming Inconsistency: Low) クラス命名において `PKSEngine`（Agent/Component）に対し `PKSBacklinks`（Noun/Data）が混在している。`PKSBacklinks` は `generate` メソッドを持つため、`BacklinkGenerator` や `BacklinkViewer` 等の Agent 的命名が適切である。
- (Verb Inconsistency: Low) 動詞の抽象度が不統一である。`push`（具体的動作）、`search`（探索）、`infer`（推論）、`suggest`（提案）、`record`（記録）、`get`（取得）が混在しており、操作の主体性が揺らいでいる（例: `auto_context_from_input` は `infer_context` であるべき）。

## 重大度
Medium
