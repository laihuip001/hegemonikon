# ハルシネーション検出者 レビュー

## 対象ファイル
`hermeneus/src/dispatch.py`

## 判定
発言（要改善）

## 発見事項
- (Critical) **標準ライブラリ `ast` のシャドウイング**:
  `hermeneus/src/dispatch.py` をスクリプトとして実行（`Usage` 通り）すると、スクリプト所在ディレクトリ `hermeneus/src` が `sys.path` の先頭に追加されます。
  これにより、同ディレクトリ内の `ast.py` が標準ライブラリの `ast` モジュールを隠蔽（シャドウイング）してしまいます。
  結果として、`dataclasses` -> `inspect` -> `ast` という依存連鎖において、標準ライブラリの代わりに `hermeneus.src.ast` が読み込まれ、循環インポートエラー（`ImportError`）が発生し、実行不可能です。

  推奨される修正: `hermeneus/src/ast.py` を `ccl_ast.py` や `nodes.py` など、標準ライブラリと競合しない名前に変更する。

## 重大度
Critical
