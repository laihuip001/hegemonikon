# ハルシネーション検出者 レビュー

## 対象ファイル
`hermeneus/src/dispatch.py`

## 判定
発言（要改善）

## 発見事項
- **Critical**: 標準ライブラリ `ast` の隠蔽（Shadowing）
    - `hermeneus/src/ast.py` というファイル名が、標準ライブラリの `ast` モジュールと競合しています。
    - `hermeneus/src/dispatch.py` を実行すると、`sys.path` の関係で `dataclasses` -> `inspect` -> `import ast` の連鎖において、標準ライブラリではなく `hermeneus/src/ast.py` が読み込まれます。
    - これにより `ImportError: cannot import name 'dataclass' from partially initialized module 'dataclasses'` (循環参照) が発生し、実行不可能です。
    - 対策: `hermeneus/src/ast.py` を `ccl_ast.py` や `nodes.py` などにリネームする必要があります。

## 重大度
Critical
