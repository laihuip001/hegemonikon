# 暗黙契約違反検出者 レビュー

## 対象ファイル
`mekhane/symploke/boot_integration.py`

## 判定
発言（要改善）

## 発見事項
- **[Medium] 軸定義の不一致 (Axis Registry Mismatch)**
  - **詳細**: `mekhane/symploke/boot_axes.py` は新たに以下の軸を定義し、`AXIS_REGISTRY` に登録していますが、`boot_integration.py` はこれらを統合していません。
    - **Axis N**: `load_violations` (Violation Trends)
    - **Axis O**: `load_gnosis_advice` (Gnōsis Advice)
  - **違反原理**: File A (`boot_axes.py`) が提供する機能を拡張（軸追加）したが、利用側 (`boot_integration.py`) が旧構成を前提としたまま更新されていません。これは「定数/設定値（利用可能な軸の集合）の変更だが参照側が対応していない」パターンに該当します。
  - **影響**: `/boot` 実行時に新しい軸（違反傾向、知識アドバイス）が表示されず、機能拡張がユーザーに届いていません。

## 重大度
Medium
