# 暗黙契約違反検出者 レビュー

## 対象ファイル
`mekhane/mcp/sympatheia_mcp_server.py`

## 判定
発言（要改善）

## 発見事項
- **プライベート関数の外部参照 (High)**
  - `_read_json`, `_load_config` および `MNEME` を `mekhane/api/routes/sympatheia.py` からインポートして使用している。これらはモジュール内部のヘルパーであり、外部公開APIではないため、変更されるとMCPサーバーが停止するリスクがある。
- **実装とドキュメントの乖離 (Medium)**
  - `sympatheia_attractor` ツールの説明に「TF-IDF ベクトル類似度」とあるが、実際の実装 (`mekhane/fep/attractor.py`, `mekhane/fep/theorem_attractor.py`) は `Embedder` (Transformer-based embedding) を使用している。
- **定数値の不正確な参照 (Medium)**
  - `sympatheia_wbc` ツールの説明に「SACRED_TRUTH.md 変更時は threatScore=15」とあるが、実際の実装 (`sympatheia.py`) では `10` (ベース) + `severity` (1, 3, 5) であり、必ずしも `15` になるとは限らない (Medium severity なら 11)。

## 重大度
High
