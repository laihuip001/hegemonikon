# 認知チャンク分析者 レビュー

## 対象ファイル
`mekhane/anamnesis/gnosis_chat.py`

## 判定
発言（要改善）

## 発見事項
- **174行目 (Medium)**: 1行に5演算以上 (split, slice, replaceなど)
  - `f.stem.split("_", 3)[-1].replace("_", " ")`
  - 複雑な演算が連鎖しており認知負荷が高い。
  - **修正案**:
    ```python
    raw_title = f.stem.split("_", 3)[-1]
    title = raw_title.replace("_", " ")
    ```
- **275行目 (Medium)**: 1行に関数呼び出しが4つ以上 (search, select, limit, to_pandas)
  - `df = table.search().select(["primary_key"]).limit(None).to_pandas()`
  - メソッドチェーンが長く、デバッグや読解が困難。
  - **修正案**:
    ```python
    query = table.search()
    query = query.select(["primary_key"])
    df = query.limit(None).to_pandas()
    ```
