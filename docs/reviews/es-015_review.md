# オンボーディング障壁検出者 レビュー

## 対象ファイル
`mekhane/symploke/jules_client.py`

## 発見事項
- **未定義のドメイン専門用語**: コード内やドキュメント文字列に「Hegemonikón」「Symplokē」「Synedrion」といったプロジェクト固有の難解な用語が説明なく使用されており、新規参加者が理解するのを困難にしています。
- **不安全なデフォルト設定**: `MAX_CONCURRENT` がデフォルトで 60 ("Ultra plan limit") に設定されています。Ultraプランを持たないユーザーが使用した場合、即座にレート制限に達する可能性があります。
- **隠れた依存関係**: `synedrion_review` メソッド内で `mekhane.ergasterion.synedrion` を動的にインポートしています。この依存関係がインストールされていない場合、セットアップ時には気づかず、実行時に予期せぬエラーが発生します。
- **威圧的な文脈情報**: 「Refactored based on 58 Jules Synedrion reviews」や「cl-003 fix」「ai-006 review」といった過去のレビューIDへの参照が頻出しますが、これらの背景情報へのアクセス手段が不明確で、新規開発者に疎外感を与えます。
- **誤解を招くハードコードされたURL**: `BASE_URL` が `https://jules.googleapis.com/v1alpha` に固定されています。これが一般に公開されているエンドポイントなのか、内部専用なのかが不明確で、接続できない場合の原因切り分けを困難にします。
- **命令的なエラーメッセージ**: `ValueError("API key required...")` などのエラーメッセージが機能的ですが不親切であり、解決方法への誘導（例：環境変数の設定方法やAPIキーの取得場所）が不足しています。

## 重大度
- High
