# 予測誤差審問官 レビュー

## 対象ファイル
`scripts/attractor_dashboard.py` (実体: `scripts/fep_dashboard.py`)

## 判定
発言（要改善）

## 発見事項
- **副作用非明示 (High)**: `sys.path.insert` によるグローバルなインポートパス変更がモジュールレベルで行われている (L19)。
  - *修正案*: `sys.path` 操作を `main` 関数内または `if __name__ == "__main__":` ブロック内に移動する。
- **型予測困難 (Medium)**: `analyze` 関数の戻り値が `dict` とされており、具体的な構造（キー名や値の型）がシグネチャから予測不能。
  - *修正案*: `TypedDict` または `dataclass` を定義して戻り値の型として使用する。
- **暗黙状態変更 (Medium)**: `main` 関数内で `Path.home()` を使用したハードコードされたパスへの依存があり、実行環境による振る舞いの変化が予測しづらい。
  - *修正案*: ログディレクトリを引数または環境変数で指定可能にし、デフォルト値を定数として分離する。
- **例外の隠蔽 (Low)**: `load_log` 関数で `json.JSONDecodeError` を握りつぶしており、データ破損時に何が起きたか不明瞭。
  - *修正案*: エラーログを出力するか、破損行数をカウントして呼び出し元に報告する。

## 重大度
High
