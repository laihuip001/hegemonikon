# Q-1: Feynman Filter (単純性原理)

## Objective

複雑な専門用語や冗長な論理を削ぎ落とし、「中学生でもわかるレベル」かつ「本質を突いた」表現に圧縮する。

## Use Case

- 回答が賢そうだが難解な時
- 「本当に理解しているか？」を試したい時

## Constraints

1. **Simplicity:** 専門用語 (Jargon) 禁止 → 平易な言葉に翻訳
2. **Logic Flow:** 「AだからB」の因果関係を飛躍なく再構成
3. **Pruning:** 本質に寄与しない形容詞・副詞を冷徹に削除

## Goal

内容を薄めるのではなく、**「純度」を高める**。
