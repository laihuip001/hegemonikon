## 5分で読む要約

### 公理階層（7軸）

```
L0: What — FEP
L1: Who/Why — Flow (I/A), Value (E/P)
L1.5: Where-When/How — Scale (M/M), Function (E/E)
L1.75: Which/How much — Valence (+/-), Precision (C/U)
```

### 定理群（24 = 6×4）

| 流れ | 記号 | 名称 | 役割 |
|------|------|------|------|
| Poiēsis | O | Ousia | 本質 |
| Poiēsis | S | Schema | 様態 |
| Poiēsis | H | Hormē | 傾向 |
| Dokimasia | P | Perigraphē | 条件 |
| Dokimasia | K | Kairos | 文脈 |
| Dokimasia | A | Akribeia | 精密 |

**X-series**: 72関係（定理群間の従属）

**総数**: 24 + 72 = **96**

---

## 日常適用早見表

### Poiēsis: 内容の具現化（生成層）

| 定理群 | 問い | 発動タイミング |
|--------|------|----------------|
| **O: Ousia** | 何をするか | 本質的な目的を問う |
| **S: Schema** | どの配置で | 様態・アプローチを決める |
| **H: Hormē** | どちらへ | 傾向と確信度を調整 |

### Dokimasia: 条件の詳細化（審査層）

| 定理群 | 問い | 発動タイミング |
|--------|------|----------------|
| **P: Perigraphē** | どの場で | 条件空間を定義 |
| **K: Kairos** | いつやるか | 文脈に応じた調整 |
| **A: Akribeia** | どの程度精密に | 最終的な微調整 |
