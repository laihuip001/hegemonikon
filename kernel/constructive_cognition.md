# 建設的認知 (Constructive Cognition) — 認知を生産的に方向づける原理

> **分類**: 認知的公理 (Cognitive Axiom)
> **導出**: FEP Function 公理 (Explore ↔ Exploit) + ストア派 Dichotomy of Control
> **宣言**: **認知の目的は、変えられるものを変え、変えられないものを受け入れ、その区別に注意資源を配分することである。**

---

## 1. ニーバーの祈りと FEP の同型性

```
God, grant me the serenity to accept the things I cannot change,
    Courage to change the things I can,
    And wisdom to know the difference.
```

| ニーバーの祈り | FEP / HGK | 操作的意味 |
|:------------|:----------|:----------|
| 変えられないものを受け入れる冷静さ | **Exploit** — 既知の制約を受容 | 構造的限界 (Layer C) を認め、そこにエネルギーを浪費しない |
| 変えられるものを変える勇気 | **Explore** — 未知に挑む行動 | 制御可能な変数に注意資源を集中する |
| 区別する知恵 | **Precision weighting** — π = V[ε]⁻¹ | どちらに注意を配分するかの精度加重 |

> **定理**: ニーバーの祈りは FEP の Function 公理 (Explore ↔ Exploit) の自然言語表現である。
> Function 公理は EFE (Expected Free Energy) による行動選択として定式化される:
>
> - Explore = 情報利得の最大化 (epistemic value)
> - Exploit = 実用的価値の最大化 (pragmatic value)
> - 区別 = 情報利得と実用的価値の精度加重 π による最適配分

## 2. ストア派ディクホトミーとマルコフブランケット

エピクテトス『手引き (Encheiridion)』:

> 「我々の力の及ぶものがあり、及ばないものがある。
> 及ぶものとは、意見、欲求、嫌悪、要するに我々の行為である。
> 及ばないものとは、身体、財産、評判、官職、要するに我々の行為でないものである。」

| ストア派 | マルコフブランケット (FEP) | HGK |
|:--------|:------------------------|:----|
| τὰ ἐφ᾿ ἡμῖν (我々に依るもの) | **内部状態** (μ) + **能動状態** (a) | 制御可能 → Explore |
| τὰ οὐκ ἐφ᾿ ἡμῖν (依らないもの) | **外部状態** (η) + **感覚状態** (s) | 制御不能 → Accept |
| 区別 | **ブランケットの境界** | Precision weighting |

> **操作的類推**: ストア派のディクホトミーはマルコフブランケットの内/外の分割と対応する。
> ただし2元素間の対応は自明。非自明な構造は下記の精緻化を参照。

### 圏論的精緻化: 操作的類推の構造的根拠

> **ステータス**: 以下は Level B (精密な比喩) であり Level A (厳密な圏論的証明) ではない。
> HGK Category Engine の分類 (SKILL.md L308) に従う。

#### 素朴な同型は空虚

§2 の対応表は {制御可能, 制御不能} ↔ {内部, 外部} の2元素間の全単射に見える。
2元素集合間の全単射は2つ (恒等と交換) しか存在しない。「正しい方を選ぶ」だけでは
数学的に空虚であり、「ガロア同型を証明した」とは言えない。

#### 非自明な構造: エピクテトスの6概念 → MB の4変数

ストア派を2元素に退化させず、エピクテトス『手引き』の原文に立ち戻る:

| ストア派概念 | 原語 | 分類 | MB 変数 | 写像の根拠 |
|:------------|:-----|:-----|:--------|:----------|
| 意見 | δόγμα | eph (制御可能) | **μ** (internal) | 信念 = 内部モデルの更新 |
| 欲求 | ὄρεξις | eph | **a** (active) | 接近行動 = 能動的状態変化 |
| 回避 | ἔκκλισις | eph | **a** (active) | 回避行動 = 能動的状態変化 |
| 身体 | σῶμα | ouk-eph (制御不能) | **η** (external) | 物理的環境 |
| 財産 | κτῆσις | ouk-eph | **η** (external) | 社会的環境 |
| 評判 | δόξα | ouk-eph | **s** (sensory) | 他者からの観測 = 感覚入力 |

**写像 L: Stoic₆ → MB₄ は非単射**

- L(ὄρεξις) = L(ἔκκλισις) = a — 欲求と回避が「能動状態」に崩壊
- L(σῶμα) = L(κτῆσις) = η — 身体と財産が「外部状態」に崩壊

この崩壊こそが**忘却関手の Drift**。ストア派は欲求と回避を倫理的に区別するが、
FEP の能動状態は符号の違い (approach/avoid) を内部パラメータとして吸収する。
同様にストア派は身体と財産を種類として区別するが、MB は「外部」として一括する。

#### Drift の測定

| 崩壊 | 失われる区別 | Drift |
|:-----|:-----------|:------|
| ὄρεξις → a ← ἔκκλισις | 接近と回避の倫理的区別 | ε < 1 — FEP は符号で表現するが、ストア派の道徳的重みは消える |
| σῶμα → η ← κτῆσις | 身体と財産の存在論的区別 | ε < 1 — FEP はどちらも「介入不能な外部」として扱う |
| δόξα → s | — | ε ≈ 1 — 一対一対応。Drift なし |
| δόγμα → μ | — | ε ≈ 1 — 一対一対応。Drift なし |

#### 主張の格付け

| 主張 | 格付け | 確信度 |
|:-----|:-------|:------|
| 「ストア派 = FEP はガロア同型」 | ❌ **棄却** | — 2元素間では空虚 |
| 「6概念→4変数は非自明な忘却写像」 | [仮説: 65%] | 順序構造の厳密定義が未完 |
| 「操作的類推として有用」 | [推定: 85%] | SKILL.md の Level B 基準で妥当 |

> **正直さ**: ストア派 ↔ FEP の対応は**操作的類推** (operational analogy) であり、
> 圏論的同型ではない。忘却が存在し、Drift > 0 である。
> この類推が有用なのは、**日常言語 (ストア派) と数学的形式 (FEP) の間を
> 双方向に行き来するための辞書**として機能するからである。

## 3. 建設的認知の定義

### 定義

> **建設的認知** (Constructive Cognition) とは、認知資源を**制御可能な変数**に集中し、
> **制御不能な変数**を受容し、その区別に精度加重を適切に配分する認知姿勢である。

### 3つの原理

| # | 原理 | FEP | 操作的行動 |
|:--|:-----|:----|:----------|
| **C-1** | **可能性の探索** (Explore) | EFE の情報利得項 | 「できるとしたら、どうやって？」を先に問う |
| **C-2** | **制約の受容** (Accept) | Layer C の認識 | 構造的限界を認め、そこに固執しない |
| **C-3** | **区別の精度** (Discern) | Precision weighting | 変えられるものと変えられないものを正確に分類する |

### 反建設的認知のパターン

| パターン | 問題 | 建設的転換 |
|:--------|:-----|:----------|
| 「できない」で止まる | C-1 違反 — 可能性の探索を放棄 | 「代わりに何ができるか？」 |
| 変えられないものに執着 | C-2 違反 — 受容の欠如 | 「これは今の制約として認める」 |
| 全てを変えようとする | C-2 違反 — 受容の欠如 | 「制御可能な部分はどこか？」 |
| 問題原因の分析に没頭 | C-3 違反 — 区別の失敗 | 「原因ではなく、次の一歩は？」 |
| 変えられるのに諦める | C-1 違反 + CD-1 (全か無か) | 「最小の行動は何か？」 |

## 4. 神話からの導出 — 建設的認知の原型

### C-1 可能性の探索

| 神話 | 英雄 | 建設的行動 | 教訓 |
|:-----|:-----|:----------|:-----|
| **トロイの木馬** | オデュッセウス | 力押しを捨て、欺瞞による視点転換 | 正面突破ではなく創造的な迂回路を探せ |
| **ゴルディアスの結び目** | アレクサンドロス | 問題の枠組み自体を再定義 (剣で切断) | ルールに縛られず本質を優先せよ |
| **アウゲイアスの厩** | ヘラクレス | 川の流れを活用して自動化 | 自然の力 (既存のシステム) を活用せよ |
| **わらしべ長者** | 名もなき男 | 小さな交換の連鎖で価値を増大 | 最小の一歩が連鎖的に大きな変化を生む |
| **ダイダロスの翼** | ダイダロス | 制約の中での発明 (蝋と羽) | 制約がイノベーションを駆動する |

### C-2 制約の受容

| 神話 | 英雄 | 受容した制約 | 教訓 |
|:-----|:-----|:-----------|:-----|
| **プロメテウスの火** | プロメテウス | 罰 (岩縛り) を受容して火を与えた | 短期の犠牲を受容し、長期の進歩を選べ |
| **オーディンの知恵** | オーディン | 片目の喪失 + ラグナロクの運命 | 変えられない運命を認め、変えられる知識追求に集中 |
| **ヒュドラの首** | ヘラクレス | 切っても再生する構造的制約 | 問題を根絶するのではなく、再発防止メカニズムを構築 |

### C-3 区別の精度

| 神話 | 知恵 | 区別 | 教訓 |
|:-----|:-----|:-----|:-----|
| **指と月** (禅) | 指 ≠ 月 | 手段と目的の区別 | 道具に執着せず、本質を見極めよ |
| **テセウスの糸** | アリアドネの糸 | 制御可能 (糸=計画) と制御不能 (迷宮=環境) | 不確実な環境でこそ、確実な手段 (トレースバック) に頼れ |
| **アリとキリギリス** | 準備の知恵 | 変えられる未来 (備蓄) と変えられない冬 (季節) | 確実に来る制約に対して、今できることに集中 |
| **キクロプス退治** | 「我が名は無体」 | 力 (制御不能) と機転 (制御可能) | 最小リソースで最大効果を生む区別 |

### 認知的公理としての位置づけ

> 建設的認知は、7公理のうち **Function** と **Precision** の合成として導出される:
>
> - Function が「何をするか」(Explore/Exploit) を決め
> - Precision が「どれだけ注意を配分するか」を決める
> - 両者の合成が「制御可能なものに精確に注意を向ける」= 建設的認知

## 6. 操作的チェックリスト

認知が非建設的になっていると感じたとき:

```
[建設的認知チェック]
1. 今、問題の原因を分析しているか、解決策を探しているか？ → 解決策へ (C-1)
2. 制御できないものに固執していないか？ → 受容せよ (C-2)  
3. 本当に制御不能か、「やっていない」だけか？ → 区別せよ (C-3)
4. 最小の一歩は何か？ → わらしべ長者の一本のわらから始めよ (C-1)
5. オデュッセウスなら、ヘラクレスならどうするか？ → 視点を変えよ
```
