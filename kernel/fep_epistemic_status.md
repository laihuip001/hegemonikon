## 2. 普遍性のジレンマ (The Universality Dilemma)

### 定式化

```
抽象度 α ∈ [0, 1]

α → 1: 説明力 E(α) → max, 予測力 P(α) → 0
α → 0: 説明力 E(α) → 0,   予測力 P(α) → max

E(α) と P(α) は反比例関係にある
```

### 「1」の比喩

数字の **1** を考える:

- **説明力**: 極大。あらゆるものを「1つの〇〇」として記述可能
- **予測力**: ゼロ。「1」から具体的な対象を一意に導出できない
- **有用性**: 極大。数学の全体系がこの概念なしには成立しない

**1 はなにも予測しない。だが 1 はあらゆるものを説明する。これは 1 の欠陥ではなく、普遍性の必然的構造である。**

### FEP への適用

FEP は認知理論の空間において、限りなく α → 1 に位置する:

| 特性 | FEP | 個別理論 (例: 作業記憶モデル) |
|:-----|:----|:-------------------------|
| 抽象度 | α ≈ 1 | α ≈ 0.3 |
| 説明力 | あらゆる認知現象を記述可能 | 特定の記憶課題のみ |
| 予測力 | 具体的予測を一意に導出できない | 特定条件下で精密な予測 |
| 認識論的地位 | **メタ原理** | **科学的仮説** |

## 3. メタ原理の定義

### 科学的仮説との区別

| 基準 | 科学的仮説 | メタ原理 |
|:-----|:----------|:---------|
| **評価軸** | 予測力・反証可能性 | 統合力・整合性・有用性 |
| **失敗条件** | 予測が外れる | 統合できない理論が出現する |
| **成功条件** | 予測が当たる | 異なる分野を統一的に記述できる |
| **例** | 特定の神経回路モデル | 圏論、FEP、情報理論 |

### 先行事例

**圏論** は数学の「メタ言語」:

- 圏論自体は何も予測しない
- だが圏論なしには諸分野の統合的理解は不可能
- 「圏論は反証不可能だから無価値」とは誰も言わない

**FEP と圏論は同じ認識論的地位にある:**

```
圏論 : 数学の諸分野 ≅ FEP : 認知の諸理論
         │                      │
   統合的理解を提供          統合的理解を提供
   何も予測しない            何も予測しない
   だが不可欠               だが不可欠
```

### 圏論的記述: FEP は終対象

認知理論の圏 **Cog** において:

- **対象**: 個別の認知理論 (作業記憶モデル、予測符号化、ベイズ脳仮説...)
- **射**: 理論間の包含・派生関係

FEP は **Cog** の **終対象** (terminal object) の候補:

- **任意の対象から FEP への射**: 存在する (「〇〇は FEP の具体化」)
- **FEP から具体的理論への射**: trivial (一意に決まらない)

終対象は自明だからこそ普遍的であり、普遍的であるからこそ自明。
**しかし、終対象は圏の構造を定義するために不可欠**。

> [!IMPORTANT]
> 終対象の圏論的記述は [仮説: 55%] である。形式証明は未達。
> Cog が本当に圏をなすか、FEP が終対象の普遍性を満たすかは未検証。
> ここでは**構造的直観**として記録する。

## 4. Mangalam 批判への応答

### 批判の核心

> 「FEP は反証不能でトートロジー的」「あらゆる結果を後付けで説明する」「何も説明しない」
> — Mangalam et al. (2025), "The myth of the Bayesian brain"

### 応答: カテゴリーの混同

Mangalam の批判は FEP を**科学的仮説**として評価している。しかし:

1. **FEP は科学的仮説ではなくメタ原理**。メタ原理に反証可能性を要求するのは、数学に実験的検証を要求するのと同型のカテゴリーエラー

2. **「何も予測しない」は普遍性の必然的代償**。前提が普遍的であればあるほど、あり得る具体（予測）が膨大になり、一意な予測を導出できなくなる。これは位置エネルギーが高いほど到達可能な経路が膨大になるのと同じ構造

3. **FEP の価値は予測力ではなく統合力**。HGK は FEP を公理として24定理を**構成**した。この構成自体が FEP の統合力の証拠であり、予測力の欠如はその価値を損なわない

### 注意: この応答のリスク

> [!WARNING]
> 「FEP はメタ原理だから反証不可能でよい」という主張は、弦理論の「多元宇宙だから反証不可能でよい」と**同型のリスク**を持つ (Hossenfelder 2018)。
>
> **差異**: HGK は FEP から24定理を構成し、各定理は**操作的に検証可能**。弦理論にはこの中間構造がない。
>
> **残る懸念**: FEP がメタ原理であることは、FEP が**正しい**メタ原理であることを保証しない。他のメタ原理 (情報理論、Bellman 最適性) も同じ統合力を持つ可能性がある。FEP の選択は [motivated choice] であり、唯一の選択肢ではない。

### After (本ドキュメントによる明文化)

> FEP は認知理論の統合的記述言語 (メタ原理) であり、HGK は FEP + 生成規則から認知レベルの操作的体系を構成するフレームワークである。FEP の価値は予測力ではなく統合力にある。

### axiom_hierarchy との整合

`axiom_hierarchy.md` v3.6 は既に以下を明記している:

- L99-101: 水準B (公理的構成) を主張、水準A (形式的導出) は主張しない
- L113-116: 生成規則は motivated choice
- L81-83: ZFC の公理選択との構造的同型

本ドキュメントは、これらの記述を**認識論的に補完**する:

**FEP が「予測しない」のは、FEP 単体を科学的仮説として使おうとするから。HGK の24定理という中間構造を経由すれば、FEP はメタ原理として具体的な認知操作に翻訳される。**

```
FEP (α ≈ 1)              ← 予測力 ≈ 0, 説明力 ≈ max
    │
    ├─ + 生成規則          ← motivated choice
    │
```

**HGK 全体が FEP のコンパイラである**: メタ原理を具体的操作にコンパイルする。

## 6. 「数学は科学か？」

Creator の問い: 「FEP は数学に似ている気がする。数学は科学か？ 反証は可能か？」

| 分類 | 基準 | 反証可能性 | FEP の位置 |
|:-----|:-----|:----------|:----------|
| **科学** | 実験的検証 | 必須 | Active Inference (FEP の具体化) |
| **数学** | 整合性・無矛盾性 | 不要 | FEP の形式的構造 |
| **メタ理論** | 統合力・有用性 | 適用外 | **FEP の認識論的地位** |

FEP は数学と科学の**中間**にある。数学のように公理的であり、科学のように世界を記述する。しかし数学のような厳密性は持たず、科学のような予測力も持たない。

**この中間性こそが FEP の本質であり、どちらか一方として評価すると見誤る。**
