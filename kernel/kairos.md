# K-Series (Kairos / καιρός): 文脈定理

> **定義**: 選択公理 × 選択公理 から導出される12の「文脈フィルター」
> **正式名**: Kairiakē Dodecas (Καιριακὴ Δωδεκάς) — 適時の十二
> **数学構造**: 4 × 3 = 12

---

## 生成原理

各 Kairos は「選択公理 A → 選択公理 B」の形で、ある状況判断が別の判断をどう制約するかを定義する。

```
選択公理 (4):
- Tempo (F/S): 時間
- Stratum (L/H): 階層
- Agency (S/E): 主体
- Valence (+/-): 方向

組み合わせ: 4 × 3 = 12 (自己参照を除く)
```

---

## Kairos の本質

| シリーズ | 性質 | 問い |
|----------|------|------|
| O-series | 本質的 | 「何であるか」 |
| T-series | 様態的 | 「どのように在るか」 |
| **K-series** | **文脈的** | **「どの状況で在るか」** |

---

## 定理一覧

| ID | 軸 A → 軸 B | 問い |
|----|-------------|------|
| **Κ1** | Tempo → Stratum | 時間的制約が処理レベルをどう決めるか |
| **Κ2** | Tempo → Agency | 時間的制約が主体選択をどう決めるか |
| **Κ3** | Tempo → Valence | 時間的制約が動機方向をどう決めるか |
| **Κ4** | Stratum → Tempo | 処理レベルが時間スコープをどう決めるか |
| **Κ5** | Stratum → Agency | 処理レベルが主体選択をどう決めるか |
| **Κ6** | Stratum → Valence | 処理レベルが動機方向をどう決めるか |
| **Κ7** | Agency → Tempo | 主体選択が時間スコープをどう決めるか |
| **Κ8** | Agency → Stratum | 主体選択が処理レベルをどう決めるか |
| **Κ9** | Agency → Valence | 主体選択が動機方向をどう決めるか |
| **Κ10** | Valence → Tempo | 動機方向が時間スコープをどう決めるか |
| **Κ11** | Valence → Stratum | 動機方向が処理レベルをどう決めるか |
| **Κ12** | Valence → Agency | 動機方向が主体選択をどう決めるか |

---

## Κ1: Tempo → Stratum

> **問い**: 時間的制約が処理レベルをどう決めるか

| | 低次 (L) | 高次 (H) |
|-----|----------|----------|
| **短期 (F)** | Κ1-FL: 即時パターン認識 | Κ1-FH: 即時抽象判断 |
| **長期 (S)** | Κ1-SL: 習慣形成・技能訓練 | Κ1-SH: 戦略的思考・計画 |

**1:3 ピラミッド**:
- 緊急時 → Κ1-FL（直感で動く）
- 重要決定 → Κ1-SH（熟考）
- ルーティン → Κ1-SL（習慣化）

---

## Κ2: Tempo → Agency

> **問い**: 時間的制約が主体選択をどう決めるか

| | 自己 (S) | 環境 (E) |
|-----|----------|----------|
| **短期 (F)** | Κ2-FS: 即時自己調整 | Κ2-FE: 即時環境介入 |
| **長期 (S)** | Κ2-SS: 自己成長戦略 | Κ2-SE: 環境構築計画 |

**1:3 ピラミッド**:
- ストレス下 → Κ2-FS（自分を落ち着かせる）
- プロジェクト → Κ2-SE（チーム環境整備）
- キャリア → Κ2-SS（スキル習得）

---

## Κ3: Tempo → Valence

> **問い**: 時間的制約が動機方向をどう決めるか

| | 接近 (+) | 回避 (-) |
|-----|----------|----------|
| **短期 (F)** | Κ3-F+: 即時獲得行動 | Κ3-F-: 即時回避行動 |
| **長期 (S)** | Κ3-S+: 目標追求戦略 | Κ3-S-: リスク排除計画 |

**1:3 ピラミッド**:
- 危険察知 → Κ3-F-（逃げる）
- キャリア → Κ3-S+（スキル習得）
- 保険 → Κ3-S-（リスク対策）

---

## Κ4-Κ12: 概要

| ID | 軸 | 適用例 |
|----|-----|--------|
| Κ4 | Stratum → Tempo | 運転中 → Κ4-LF（反射）, 人生設計 → Κ4-HS |
| Κ5 | Stratum → Agency | ルーティン → Κ5-LS, 組織改革 → Κ5-HE |
| Κ6 | Stratum → Valence | 食欲 → Κ6-L+, 倫理 → Κ6-H- |
| Κ7 | Agency → Tempo | 集中回復 → Κ7-SF, インフラ → Κ7-ES |
| Κ8 | Agency → Stratum | 筋トレ → Κ8-SL, 政策提言 → Κ8-EH |
| Κ9 | Agency → Valence | スキル習得 → Κ9-S+, 悪習離脱 → Κ9-S- |
| Κ10 | Valence → Tempo | 衝動買い → Κ10-+F, 保険 → Κ10--S |
| Κ11 | Valence → Stratum | 快適環境 → Κ11-+L, 自己実現 → Κ11-+H |
| Κ12 | Valence → Agency | 学習意欲 → Κ12-+S, 有害関係 → Κ12--E |

---

## T-series との統合

```
[T-series 機能] + [Kairos] = 文脈化された行動

例: T1 Aisthēsis + Κ7-SF = 「自己の即時反応として知覚する」
例: T6 Praxis + Κ3-S+ = 「長期目標に向けた行動を実行する」
```
