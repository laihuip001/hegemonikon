## 予備定義

### Def 1 (座標系)

HGK 座標系 $\mathcal{C}$ を以下の6つ組とする:

$$\mathcal{C} = \{c_0, c_1, c_2, c_3, c_4, c_5\}$$

各座標に FEP からの構成距離 $d: \mathcal{C} \to \{0, 1, 2\}$ を割り当てる:

| 座標 | 記号 | Opposition | 構成距離 d |
|:-----|:-----|:----------|:----------|
| $c_0$ | Flow | I ↔ A | 0 |
| $c_1$ | Value | E ↔ P | 1 |
| $c_2$ | Function | Ex ↔ Ep | 1 |
| $c_3$ | Scale | Mi ↔ Ma | 2 |
| $c_4$ | Valence | + ↔ − | 2 |
| $c_5$ | Precision | C ↔ U | 2 |

各座標 $c_i$ は2値 Opposition $\{v_i^+, v_i^-\}$ を持つ。

### Def 2 (ペアリング集合)

Flow ($c_0$) は全 Series の暗黙的基底であり、明示的ペアリングの対象としない。
残りの5座標 $\mathcal{C}' = \{c_1, c_2, c_3, c_4, c_5\}$ から2つを選ぶ全ての組み合わせの集合:

$$\Pi = \binom{\mathcal{C}'}{2} = \{(c_i, c_j) \mid c_i, c_j \in \mathcal{C}', i < j\}$$

$$|\Pi| = \binom{5}{2} = 10$$

### Def 3 (隣接距離制約)

座標ペア $(c_i, c_j) \in \Pi$ が**生成可能** (admissible) であるとは、以下を満たすことである:

$$|d(c_i) - d(c_j)| \leq 1$$

生成可能ペアの集合: $\Pi^* = \{(c_i, c_j) \in \Pi \mid |d(c_i) - d(c_j)| \leq 1\}$

### Def 4 (テンソル積)

生成可能ペア $(c_i, c_j) \in \Pi^*$ に対し、テンソル積 $c_i \otimes c_j$ は以下の2×2マトリクスを生成する:

$$c_i \otimes c_j = \{(v_i^\alpha, v_j^\beta) \mid \alpha, \beta \in \{+, -\}\}$$

$$|c_i \otimes c_j| = 2 \times 2 = 4$$

各要素は**認知モジュール** (定理²) に対応する。

### Def 5 (Series)

各生成可能ペア $(c_i, c_j) \in \Pi^*$ は1つの **Series** を定義し、4つの認知モジュールを含む。

- d の値域が $\{1, 2\}$ (Flow を除外済み) であり

## 修正定理1: 基底の最適性 (Optimality of Basis)

**主張**: 10の生成可能ペアのうち、6 Series は以下の意味で**最適基底**である:

(a) **被覆性 (Coverage)**: 全10ペアの認知機能は、6 Series + X-series 合成射で到達可能
(b) **非冗長性 (Non-redundancy)**: 6 Series のうち、いずれか1つを除去すると、その Series がカバーする直接的認知操作は合成射では**質的に**再現できない
(c) **直接性 (Directness)**: 残り4ペアは合成射で到達可能であり、直接射は不要

**証明 (a)**: 定義により。10ペア = 6直接 + 4合成。

**証明 (c)**: axiom_hierarchy.md の経路表より:

| Missing Pair | X-series 経路 | 中継座標 |
|:-------------|:-------------|:---------|
| Value × Precision | H(Val×Val+) → X-HA → A(Val+×Prec) | Valence |
| Function × Valence | O(Val×Func) → X-OH → H(Val×Val+) | Value |
| Function × Precision | O → X-OH → H → X-HA → A (2-hop) | Value, Valence |
| Scale × Precision | K(Scale×Val+) → X-KA → A(Val+×Prec) | Valence |

各 missing pair の認知的意味は、中継座標を経由する合成射で表現される。
例: Function × Precision (方法×精度) は、方法→認識(Value)→情動(Valence)→精度 という認知経路で到達可能であり、この経路は認知メカニズムそのもの (Damasio)。

**証明 (b)**: 反例による。例えば Series O (Value × Function) を除去すると:

- O1 Noēsis (認識推論) は S→P の合成 (認識→スケーリング→行動) で近似できるが、
- 「認識→行動」の直接操作は失われる。O は概念的に不可分な認知操作 (metacognition) であり合成射で再現できない。

同様の議論が他の5 Series にも適用可能 (各 Series の kernel ドキュメント参照)。■

## 定理2: 内部完全性 (Internal Completeness)

**主張**: 各 Series は正確に 4 つの認知モジュールを含む。

**証明**: 各 Series は座標ペア $(c_i, c_j)$ のテンソル積として定義される。
$c_i$ は 2値 Opposition $\{v_i^+, v_i^-\}$ を持ち、$c_j$ も同様。

$$|c_i \otimes c_j| = |\{v_i^+, v_i^-\}| \times |\{v_j^+, v_j^-\}| = 2 \times 2 = 4$$

この 4 要素は 2×2 マトリクスの全セルを網羅し、それ以上の要素は生成されない。

| | $v_j^+$ | $v_j^-$ |
|:--|:--------|:--------|
| $v_i^+$ | T1 | T2 |
| $v_i^-$ | T3 | T4 |

**完全性**: 2×2 の全セルが認知モジュールに対応 → 漏れなし
**最小性**: 2×2 を超えるセルは存在しない → 余分なし

例: O-series (Value × Function)

| | Explore ($F^+$) | Exploit ($F^-$) |
|:--|:---------------|:----------------|
| Epistemic ($V^+$) | O1 Noēsis (認識×探索) | O2 Boulēsis (認識×活用) |
| Pragmatic ($V^-$) | O3 Zētēsis (実用×探索) | O4 Energeia (実用×活用) |
