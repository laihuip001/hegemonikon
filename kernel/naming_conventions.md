## 1. フレームワーク名

### Hegemonikón (Ἡγεμονικόν)

| 項目 | 内容 |
|------|------|
| **意味** | 統率中枢・指導的部分 |
| **語源** | ἡγεμών (指導者) + -ικόν (〜に関する) |
| **哲学的背景** | ストア派心理学における「魂の支配的部分」 |
| **設計意図** | FEP に基づく認知制御の「中枢」を表現 |

**ストア派での用法**:
Hegemonikón は人間の魂における理性的・統御的な部分を指す。感覚印象を受け取り、判断し、行動を指令する。これは FEP における「予測誤差最小化」の司令塔に相当する。

| **意味** | 自由エネルギー原理 |

| **Flow** | 流動 | I (推論) ↔ A (行為) | 認知の「方向」 |

|------|------|--------|---------|

| 名称 | 意味 | 対立項 | 設計意図 |

## 3. 定理群名（6シリーズ）

### O-series: Ousia (Οὐσία)

| 項目 | 内容 |
|------|------|
| **意味** | 本質・存在・実体 |
| **哲学的背景** | アリストテレス形而上学の核心概念 |
| **生成規則** | L1 × L1 = 核心 × 核心 |
| **設計意図** | 「本質的機能」— 何であるか |

### S-series: Schema (Σχῆμα)

| 項目 | 内容 |
|------|------|
| **意味** | 図式・配置・形 |
| **哲学的背景** | アリストテレス『分析学』における論理的配置 |
| **生成規則** | L1 × L1.5 = 核心 × Tier1 |
| **設計意図** | 「様態表現」— どの配置で現れるか |

**採用理由**: 旧名 Tropos (様態) は旧構造と混同の恐れ。Schema は「要素の関係的配置」を意味し、様態表現に最適。

### H-series: Hormē (Ὁρμή)

| 項目 | 内容 |
|------|------|
| **意味** | 衝動・推進力・意図 |
| **哲学的背景** | ストア派心理学の核心概念 |
| **生成規則** | L1 × L1.75 = 核心 × Tier2 |
| **設計意図** | 「方向性」— どちらへ向かうか |

**採用理由**: ストア派の行為理論における「理性的推進力」。FEP の Active Inference に直結。

### P-series: Perigraphē (Περιγραφή)

| 項目 | 内容 |
|------|------|
| **意味** | 境界・輪郭・定義域 |
| **哲学的背景** | アリストテレス『分析学』における限界付与 |
| **生成規則** | L1.5 × L1.5 = Tier1 × Tier1 |
| **設計意図** | 「条件空間」— 境界を定義 |

**採用理由**: 「周囲を描く」(περί + γράφω) という語源が条件空間の限定を完璧に表現。

### K-series: Kairos (Καιρός)

| 項目 | 内容 |
|------|------|
| **意味** | 好機・適時・正しい時 |
| **哲学的背景** | ストア派倫理学における適切な行為 |
| **生成規則** | L1.5 × L1.75 = Tier1 × Tier2 |
| **設計意図** | 「文脈」— いつ・どの状況で適切か |

### A-series: Akribeia (Ἀκρίβεια)

| 項目 | 内容 |
|------|------|
| **意味** | 精密・正確・厳密 |
| **哲学的背景** | アリストテレスの学問論 |
| **生成規則** | L1.75 × L1.75 = Tier2 × Tier2 |
| **設計意図** | 「精密調整」— 最終的な微調整 |

- 実装: Jules APIを介した866人の専門家によるコードレビュー
- 沈黙原理: 「全員が沈黙する = 品質が良い」

| 原則 | 内容 |

| **意味的正確性** | 定理・機構の役割を正確に反映 |
| **記号整合性** | シリーズ記号と頭文字の対応 |
| **響きの美しさ** | 発音しやすく、記憶に残る |
| **一貫性** | 表記規則の統一 |
