# 洗練論 (On Refinement)

> **洗練とは、必然的な構造を発見すること**
> Refinement is discovering the necessary structure.

---

## 公理

**FEP (予測誤差最小化)** から導出:

1. **必然性** — すべての要素に存在理由がある
2. **演繹性** — すべての構造は公理から導かれる
3. **収束性** — すべての選択には唯一の解がある

---

## 美の定義

```
美しいもの:
  - 存在しなければならない理由がある
  - 他のものに置き換えられない
  - 削ると全体が崩壊する

醜いもの:
  - 偶然そこにある
  - 妥協の産物である
  - 雑多に蓄積されている
```

### 定量的定義

```
美しさ = 演繹可能性 / 表現コスト
       = 展開の運動量 / 起点のトークン数
```

> **必然（演繹）の最大化 ≒ 予測誤差の最小化**

---

## エネルギー比喩

| 概念 | 物理的比喩 | 認知的意味 |
|:-----|:----------|:-----------|
| 公理 | 位置エネルギー | 圧縮された関係 |
| 演繹 | 落下運動 | 展開・導出 |
| 定理 | 運動エネルギー | 導出された知識 |
| 美しさ | エネルギー密度 | 演繹量/トークン数 |

---

## 値 vs 関係

| 種類 | 例 | 演繹可能性 | 美しさ |
|:-----|:---|:-----------|:-------|
| 値の圧縮 | π = 3.14159... | ❌ 終点 | なし |
| 関係の圧縮 | e^(iπ) + 1 = 0 | ✅ 起点 | あり |

**π はなぜ美しくないか**:

- 各桁の情報量は減少する（対数的収束）
- π 自体からは何も演繹できない
- 有限の情報を文字で抽象化しただけ

**オイラーの公式はなぜ美しいか**:

- 7トークンから無限の帰結
- 数学の5大定数 + 3大演算を統一

## 洗練 vs 削減

| 洗練 (Refinement) | 削減 (Reduction) |
|:------------------|:-----------------|
| 必然的構造の発見 | 恣意的な切り取り |
| 密度の最大化 | 情報のロス |
| 演繹 | 判断 |
| 発見 | 発明 |

> **「十分を目指すと不足に終わる。過剰を目指すと十分に到達する。」**

---

## 実践原則

### 問うべき問い

1. **これは存在しなければならないか？** (必然性)
2. **これは公理から導けるか？** (演繹性)
3. **これ以外の解はないか？** (収束性)

### 禁止事項

- ✗ 「こうしたい」という恣意性
- ✗ 「これで十分」という妥協
- ✗ 「後で整理する」という先延ばし
