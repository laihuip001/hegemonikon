## 各定理の詳細

### P1: Khōra (空間)

> **「条件空間そのもの」**

- Scale が Scale と交差
- 認知の「場」を定義
- **プラトン『ティマイオス』**: 存在が現れる「受容体」
- 例: 問題空間の定義、作業領域の設定

### P2: Hodos (経路)

> **「条件空間内の方向」**

- Scale が Function と交差
- 空間内の「道筋」を決定
- 語源: méthodos (方法) の後半
- 例: 解決への経路設計、ナビゲーション

### P3: Trokhia (軌跡)

> **「条件空間内のパターン」**

- Function が Scale と交差
- 活動の「軌道」を記録
- 例: 行動パターンの追跡、履歴分析

### P4: Tekhnē (技術)

> **「条件空間内の可能性」**

- Function が Function と交差
- 実現可能な「技法」を定義
- 例: 利用可能な手法の一覧、ツールセット
