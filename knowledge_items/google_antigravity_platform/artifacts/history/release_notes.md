# Antigravity Release Notes

## v1.16.5 (2026-02-03)

### Performance Improvements

- **Agent Manager**: Increased speed of populating @-mention search results. This is a critical enhancement for multi-agent coordination, reducing latency when referencing specific agents or context items.
- **General**: Various bug fixes and performance enhancements across the platform.

### Feature Changes

- **"Secure Mode" Rename**: The feature previously known as "Secure Mode" has been renamed. (Note: Watch for impacts on existing security protocols or prompt-based triggers referencing the old name).
