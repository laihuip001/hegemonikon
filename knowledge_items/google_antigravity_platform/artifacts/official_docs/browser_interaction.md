# Browser Interaction

## Browser Control

Antigravity controls a Chrome browser via a specialized subagent.

- **Chrome Profile**: The agent uses an isolated, separate browser profile to protect personal cookies and accounts.
- **Settings**: Users can specify the path to Chrome if it's not detected automatically and change the profile storage location.

---

## Chrome Extension

The **Antigravity Browser Extension** is required for full agent-web interaction.

- **Function**: Enables the agent to read/act on pages.
- **Maintenance**: Manual installation and troubleshooting steps are provided in the official docs for when automatic setup fails.
