# Editor Interface

## The Editor

The entry point based on VS Code, optimized for AI-native development.

---

## Tab & Navigation

- **Supercomplete**: File-wide completions.
- **Tab-to-Jump**: Logical navigation within the code.
- **Tab-to-Import**: Automatically suggestive of dependency imports.

---

## Command

Trigger natural language requests for inline code completions or terminal commands using `CMD/CTRL + I`.

---

## Agent Side Panel

The primary interaction point within the Editor for:

- Ongoing conversations
- Attaching images/context
- Switching agent modes or models
- Reviewing file diffs and commenting on changes.
