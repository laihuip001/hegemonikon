# Nemo Actions Setup

Nemo is the default file manager for the Cinnamon desktop environment. Nemo Actions allow users to add custom commands to the context menu (right-click).

## 1. "Copy Path" Action

This action allows you to copy the full path of a selected file or folder to the clipboard with a single click.

### Configuration File

Create the file `~/.local/share/nemo/actions/copy_path.nemo_action` (or `/usr/share/nemo/actions/` for all users).

```ini
[Nemo Action]
Name=パスをコピー
Comment=選択したファイルのフルパスをクリップボードにコピーします
Exec=sh -c "readlink -n -f '%F' | xclip -selection clipboard"
Icon-Name=edit-copy
Selection=notnone
Extensions=any
```

### Dependencies

- **xclip**: Required for clipboard interaction.

  ```bash
  sudo apt install xclip
  ```

### Usage

1. Right-click any file or folder in Nemo.
2. Select "パスをコピー" (Copy Path).
3. The absolute path is now in your clipboard.

### Troubleshooting

If the menu item does not appear:

- Ensure the file ends with `.nemo_action`.
- Restart Nemo with `nemo -q`.
- Check if `Selection=notnone` is applied (requires at least one file to be selected).
