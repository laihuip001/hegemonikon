はじめに：ralph-loopは「Claudeを止めずに回す」ための実務ツールですClaude Code は、普通に使うと「指示 → 1回の実行 → 応答で終了」という単発実行になりがちです。一方で現場では、次のような作業を「収束するまで繰り返してほしい」場面が多くあります。

lint / format を全部直して、再実行して、ゼロになるまで詰める
