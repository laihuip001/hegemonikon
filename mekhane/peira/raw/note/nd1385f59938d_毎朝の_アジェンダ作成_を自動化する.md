# 毎朝の「アジェンダ作成」を自動化する

> **Source**: https://note.com/tasty_dunlin998/n/nd1385f59938d
> **Published**: 2025-12-21T16:36:19+09:00
> **Collected**: 2026-02-06T21:35:28.235513

---

Sheets → Docs →（Gmail下書き／Chat通知）を、ノーコード〜ローコードで回す3つの設計パターン
会議前に、だいたい毎回やっている作業があります。スプレッドシートに散らばった「今日の議題」を拾う

体裁を整えて Google Docs に貼る

チームに共有する（メール下書き／チャット通知）

この“準備の定型作業”を、毎朝自動で回せると、地味に効きます。
手作業の「抜け漏れ」と「やり直し」も減ります。

この記事では、Google Workspace の標
