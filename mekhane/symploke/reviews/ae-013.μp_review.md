# シンプリシティの門番 レビュー

## 対象ファイル
`mekhane/ergasterion/digestor/pipeline.py`

## 判定
発言（要改善）

## 発見事項
- **Unused Import (Low)**: `from dataclasses import field` は定義されているが使用されていない。
- **Unused Variable / Dead Code (Medium)**: CLI引数 `args.dry_run` はパースされるが、`pipeline.run` の引数には `not args.execute` が渡されており、`args.dry_run` の値は無視されている。
- **Unnecessary Complexity / Confusing Interface (Medium)**: `--dry-run` オプション定義において `action="store_true"` と `default=True` が併用されており、フラグ有無に関わらず常に `True` となる（実質的に無効化できない）。また、ロジック上も無視されているため削除すべき。
- **Redundant Local Import (Low)**: `import pickle` が `_load_existing_keys` と `_deduplicate_by_similarity` の両方でローカルインポートされている。クラスレベルまたはモジュールレベルでのインポートに統一すべき。
- **Unused Function (Medium)**: `mark_as_processed` はこのファイル内および主要な呼び出し元（`scheduler.py`）から参照されておらず、パイプラインの責務（生成）とも異なる（消費）。外部依存がない場合は削除すべき。

Medium
