# 比喩一貫性の詩人 レビュー

## 対象ファイル
`mekhane/mcp/ochema_mcp_server.py`

## 判定
発言（要改善）

## 発見事項
- (比喩混在) `Ochēma` (乗り物) や `Trajectory` (軌道) という物理・乗り物の比喩体系の中で、`Context Rot` (文脈の腐敗) という生物学的・庭園の比喩が混入している (Medium)
- (抽象度の不統一) `session_info` ツールハンドラ内で `_get_ls_client` (private) や `_rpc` (raw API) を直接操作しており、他のハンドラ (`ask`, `status` 等) が `OchemaService` に委譲しているのと比較して抽象度が統一されていない (Medium)
- (命名規則不統一) MCP Tool の命名において、動詞 (`ask`, `ask_cortex`) と名詞 (`status`, `models`, `cortex_quota`, `session_info`) が混在している (Low)
- (動詞不統一) コメントや説明文において、情報の取得を表す語として "Get" (`cortex_quota`, `status`) と "Fetch" (`Fetch all trajectories`) が混在している (Low)

## 重大度
Medium
