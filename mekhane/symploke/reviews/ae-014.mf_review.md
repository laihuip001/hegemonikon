# 比喩一貫性の詩人 レビュー

## 対象ファイル
`mekhane/anamnesis/session_indexer.py`

## 判定
発言（要改善）

## 発見事項
- **[Metaphor Mixed] (Medium)**: 同一モジュール内で「ギリシャ哲学/神秘主義」（Gnosis, Anamnesis, Hegemonikon, Mneme, Oikos）、「SF/物理」（Antigravity, Trajectory）、「生物学」（Brain）、「技術/工場」（Indexer, Session, Factory-like processing）の比喩が混在している。
    - 特に `_BRAIN_DIR` (生物) と `GnosisIndex` (神秘) の同居、`Antigravity` (SF) の唐突な出現が世界観を破壊している。
    - `GnosisIndex` (神秘) と `Embedder` (機能) の抽象度と世界観のギャップ。
- **[Naming Inconsistency] (Low)**: `fetch_all_conversations` (fetch) と `parse_sessions_from_json` (parse/load) で、データ取得・変換の動詞がコンテキストによって揺れている（APIからは fetch、ファイルからは parse だが、本質は `ingest` や `load` で統一可能）。
- **[Naming Inconsistency] (Low)**: `sessions` (概要) と `conversations` (詳細) の用語の使い分けが曖昧。どちらも「対話の記録」であり、`trajectories` という用語も混ざっている。
