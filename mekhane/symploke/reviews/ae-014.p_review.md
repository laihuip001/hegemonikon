# 比喩一貫性の詩人 レビュー

## 対象ファイル
`mekhane/ochema/service.py`

## 判定
発言（要改善）

## 発見事項
- **Metaphor Mix (Medium)**: `Ochēma` (乗り物/物理) ドメインにおいて、`Cortex` (脳/生物) や `context_health` (健康/生物)、`thinking_budget` (思考/認知) といった生物学的・庭園的メタファーが混在している。特に `Antigravity` (反重力/物理) と `Cortex` の対比は、機械と生物の不協和音を生んでいる。
- **Abstraction Level Mismatch (Medium)**: 統一サービス層 (`Unified LLM Service`) であるにもかかわらず、`session_info`, `context_health`, `ls_models` といった特定のバックエンド (`Antigravity/LS`) に依存したメソッドが、`ask`, `stream` といった抽象化されたメソッドと同列に公開されている。
- **Naming/Verb Inconsistency (Low)**: メソッド命名において、`ask`, `stream`, `get` (動詞) と `models`, `status`, `quota` (名詞) が混在している。また、内部呼び出し先で `quota_status` (状態) と `retrieve_quota` (動作) が不統一であり、それを `quota` (名詞) としてラップしているが、一貫した動作動詞 (e.g., `get_quota`) に統一されていない。

## 重大度
Medium
