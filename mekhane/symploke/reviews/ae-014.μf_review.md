# 比喩一貫性の詩人 レビュー

## 対象ファイル
`mekhane/ochema/cortex_client.py`

## 判定
発言（要改善）

## 発見事項
- **Abstraction Dissonance (Medium)**: `ask` (Driver's Intent) coexists with `load_code_assist` (Engine Wiring). The method `load_code_assist` leaks the raw API endpoint name (`:loadCodeAssist`) into the high-level client interface. In `Ochēma`, we should wrap the implementation details. Sibling `AntigravityClient` uses `get_status`, not `rpc_get_status`. Suggest `get_project_info` or `get_status`.
- **Verb Semantics Overload (Low)**: The verb `load` is bisected. `_load_oauth_config` reads a local file (IO), while `load_code_assist` performs a remote RPC call (Network). One verb, two distinct mechanical actions.
- **Naming Pattern Violation (Low)**: `retrieve_quota` uses `retrieve` (uncommon in module), whereas `_get_token` uses `get`. Sibling `AntigravityClient` uses `quota_status`. Suggest unifying to `get_quota` or `quota_status`.
