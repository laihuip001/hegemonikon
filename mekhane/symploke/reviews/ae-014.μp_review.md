# 比喩一貫性の詩人 レビュー

## 対象ファイル
`mekhane/mcp/ochema_mcp_server.py`

## 判定
発言

## 発見事項
- **Metaphor Dissonance (Medium)**: `Context Rot` (Organic/Decay) vs `Ochēma` (Vehicle/Mechanism).
  - Context: `session_info` tool description and logic uses "Context Rot" to describe state degradation.
  - Critique: In the domain of `Ochēma` (Vehicle), degradation should be described as "Drift" (navigational deviation) or "Wear" (mechanical degradation), rather than "Rot" (biological decay).
- **Naming Inconsistency (Low)**: Asymmetry in tool names `ask` vs `ask_cortex`.
  - Context: `ask` routes via LS, `ask_cortex` routes directly.
  - Critique: `ask` hides the destination, while `ask_cortex` exposes it. `ask_ls` vs `ask_cortex` (Destination-based) or `ask_proxy` vs `ask_direct` (Method-based) would be consistent.
- **Metaphor Dissonance (Low)**: `Cascade` vs `Trajectory`.
  - Context: `Cascade ID` and `GetAllCascadeTrajectories`.
  - Critique: `Cascade` (Vertical flow/Waterfall) conflicts with `Trajectory` (Horizontal/Vector motion). "Cascade" implies gravity-fed flow, while "Trajectory" implies ballistic or propelled motion.
- **Abstraction Leakage (Medium)**: Direct private member access violating the Service boundary.
  - Context: `session_info` calls `svc._get_ls_client()._rpc(...)`.
  - Critique: The MCP server (Interface) is bypassing the Service (Engine) abstraction layer. It should call `svc.get_session_info()` or similar. This breaks the "Form Follows Function" principle by exposing internal wiring in the interface layer.

## 重大度
Medium
