## 発見事項
- **存在しないモジュールバージョン (Critical)**: `requirements.txt` に記載されている `mcp==1.26.0` は、Anthropic MCP SDK の現行バージョン (v1.2.x系) と乖離しており、存在しない可能性が高い（あるいは未来のバージョンを幻覚している）。
- **存在しないモジュール (Critical)**: 以下のインポートは標準的な `mcp` SDK 構造に含まれていない可能性が高い。
    - `mcp.server.transport_security`
    - `mcp.server.auth.provider`
    - `mcp.shared.auth`
    - `mcp.server.auth.settings`
- **存在しないメソッド/引数 (Critical)**: `FastMCP` クラスのコンストラクタにおいて、以下の引数は標準APIには存在しない可能性が高い。
    - `auth_server_provider`
    - `transport_security`
    - `auth`
- **存在しない引数値 (High)**: `mcp.run(transport="streamable-http")` は標準的なトランスポート ("sse", "stdio") ではない。
- **存在しないクラス (High)**: `OAuthAuthorizationServerProvider`, `OAuthClientInformationFull` などの高度な認証クラス群は、現在の MCP Python SDK には実装されていない可能性が高い。
