# Series適所配置検査官 レビュー

## 対象ファイル
`mekhane/anamnesis/session_monitor.py`

## 判定
発言（要改善）

## 発見事項
- **Series不一致 (Medium)**: 本ファイルは `AntigravityClient` (LS API) のポーリングとデーモンループという「統合・オーケストレーション」の責務を持っています。これは `anamnesis` (記憶・永続化) ではなく、`symploke` (統合・接続) の **S-series** の責務です。`anamnesis` は受動的なストレージ機能に留まるべきです。
- **責務の越境 (Low)**: `format_session_md` 関数によるMarkdown整形ロジックが含まれています。これは出力生成 (`poiema`) やワークショップ (`ergasterion`) の領域に近く、純粋な記憶の責務を超えています。
- **提案**: 本ファイルを `mekhane/symploke/session_monitor.py` に移動し、`anamnesis` は単に保存用 API (`save_session` 等) を提供する形にリファクタリングすることを推奨します。

## 重大度
Medium
