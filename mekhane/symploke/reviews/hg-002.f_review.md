# 予測誤差審問官 レビュー

## 対象ファイル
`mekhane/periskope/models.py`

## 判定
発言（要改善）

## 発見事項
- **責務の越境と予測困難な戻り値**: `PeriskopeReport.citation_health` プロパティが、データモデル内にプレゼンテーションロジック（絵文字 "🟢" 等）を含んでいます。純粋なデータ構造であるべき `models.py` が UI 表現を返すことは呼び出し元にとって「驚き」であり、プログラム的な判定（例: `if health == ...`）を困難にします。
    - **重大度**: Medium
    - **提案**: `citation_health` は `Enum` またはステータス識別子を返し、表示用文字列は `display_citation_health` 等の別プロパティや UI 層に委譲すべきです。

- **マジックナンバーの使用**: `citation_health` 内で `0.1`, `0.3` という閾値が直接記述されており、これらの値の意味や由来が外部から不明瞭です。
    - **重大度**: Low
    - **提案**: `TAINT_TOLERANCE_LOW`, `TAINT_TOLERANCE_HIGH` などの定数として定義すべきです。
