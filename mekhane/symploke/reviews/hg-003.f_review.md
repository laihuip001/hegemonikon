# ストア派制御審判 レビュー

## 対象ファイル
`mekhane/periskope/models.py`

## 判定
発言（要改善）

## 発見事項
- **タイムアウト設定の欠如 (High)**: `PeriskopeConfig` に外部 API (SearxNG, Exa, Gemini) 呼び出しに対するタイムアウト設定が存在しない。外部サービスが応答しない場合、システムが無限に待機するリスクがある。
  ```python
  @dataclass
  class PeriskopeConfig:
      # ... (existing fields)
      # Control config
      timeout_seconds: float = 30.0  # Default timeout
  ```
- **再試行・バックオフ設定の欠如 (Medium)**: ネットワークの一時的な障害に対する再試行回数やバックオフ戦略を制御する設定が含まれていない。
  ```python
  @dataclass
  class PeriskopeConfig:
      # ... (existing fields)
      max_retries: int = 3           # Default retry count
      retry_backoff: float = 1.0     # Backoff factor
  ```

## 重大度
High
