## 覚えなくていい

**FEP 派生選択器**が状況を見て自動で選ぶ。

```
/noe  → 自動で nous/phro/meta を選択
/bou  → 自動で desir/voli/akra を選択
```

手動で指定したいときだけ:

```
/noe@nous    または
/noe --mode=nous
```
