## 1. 目的

Hegemonikón の使い方を視覚的に学べるインタラクティブ Web アプリケーションを構築する。

### 成果物

1. **デモ動画** (browser_subagent で録画)
2. **インタラクティブガイド** (Web UI)

---

## 2. 技術スタック

| レイヤー | 技術 | 理由 |
|:---------|:-----|:-----|
| **フレームワーク** | Vite + Vanilla JS | 軽量、高速、依存少 |
| **スタイル** | Vanilla CSS | TailwindCSS 不使用 (GEMINI.md 準拠) |
| **デザイン** | ダークモード + グラスモーフィズム | Premium 感 |
| **フォント** | Inter (Google Fonts) | モダン、可読性高 |
| **アニメーション** | CSS Transitions + 微動アニメーション | 上質な UX |

## 7. 実装手順

### Step 1: プロジェクト初期化

```bash
cd /home/laihuip001/oikos
mkdir -p hegemonikon-guide
cd hegemonikon-guide
npx -y create-vite@latest ./ --template vanilla
npm install
```

### Step 2: CSS 設計

1. `index.css` に上記デザイントークンを定義
2. グラスモーフィズムの `.glass-card` クラス作成
3. Hub カラーの `.hub-o`, `.hub-s` 等作成
4. ホバーアニメーション追加

### Step 3: データ準備

1. `data/` ディレクトリに JSON ファイル作成
2. 既存ドキュメントからデータ抽出

### Step 4: 画面実装

1. ホーム画面
2. Hub ナビゲーション
3. CCL 演算子ページ
4. X-series 可視化
5. インタラクティブ入力

### Step 5: 動作確認

```bash
npm run dev
```
