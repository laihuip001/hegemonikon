# Hegemonikón クイックスタート

> **5分で今日何をすべきか決める**

---

## 1. 今日最初にやること

```
/boot
```

これだけ。システムが前回の続きを教えてくれる。

---

## 2. 今日何をしたいか決める

```
/bou-
```

1行で「今日の目的」が出る。

---

## 3. よく使う4パターン

| したいこと | コマンド |
|:-----------|:---------|
| **深く考える** | `/noe+` |
| **計画を立てる** | `/s+` |
| **実行する** | `/ene+` |
| **検証する** | `/dia` |

---

## 4. 迷ったら

```
/bou+~/zet
```

「何をしたいか」と「何を問うべきか」を往復する。

---

## 5. 終わるとき

```
/bye
```

引き継ぎが自動生成される。次回の `/boot` で復元。

---

## まとめ

```
朝: /boot → /bou-
昼: /noe+ か /s+ か /ene+
夜: /bye
```

**以上。5分終了。**
